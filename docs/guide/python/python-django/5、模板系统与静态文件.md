# 第5章：模板系统与静态文件

本章学习 Django 模板语言、模板继承、静态文件管理和媒体文件处理。

---

## 5.1 模板基础

### 变量

```html
<!-- 基本变量 -->
<h1>{{ title }}</h1>
<p>{{ user.username }}</p>

<!-- 过滤器 -->
<p>{{ name|upper }}</p>
<p>{{ date|date:"Y-m-d" }}</p>
<p>{{ text|truncatewords:30 }}</p>
<p>{{ value|default:"N/A" }}</p>
```

### 标签

```html
<!-- if 语句 -->
{% if user.is_authenticated %}
    <p>欢迎, {{ user.username }}</p>
{% else %}
    <a href="{% url 'login' %}">登录</a>
{% endif %}

<!-- for 循环 -->
<ul>
{% for post in posts %}
    <li>{{ post.title }}</li>
{% empty %}
    <li>暂无文章</li>
{% endfor %}
</ul>

<!-- URL 反向解析 -->
<a href="{% url 'post-detail' pk=post.pk %}">{{ post.title }}</a>

<!-- 静态文件 -->
{% load static %}
<img src="{% static 'images/logo.png' %}">
<link rel="stylesheet" href="{% static 'css/style.css' %}">
```

### 注释

```html
{# 单行注释 #}

{% comment %}
多行注释
{% endcomment %}
```

## 5.2 模板继承

### 基础模板

```html
<!-- base.html -->
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}我的网站{% endblock %}</title>
    {% load static %}
    <link rel="stylesheet" href="{% static 'css/base.css' %}">
    {% block extra_css %}{% endblock %}
</head>
<body>
    <header>
        <nav>
            <a href="{% url 'index' %}">首页</a>
            <a href="{% url 'post-list' %}">文章</a>
        </nav>
    </header>
    
    <main>
        {% block content %}{% endblock %}
    </main>
    
    <footer>
        <p>&copy; 2024 我的网站</p>
    </footer>
    
    {% block extra_js %}{% endblock %}
</body>
</html>
```

### 子模板

```html
<!-- post_list.html -->
{% extends "base.html" %}

{% block title %}文章列表 - {{ block.super }}{% endblock %}

{% block content %}
<h1>文章列表</h1>

<div class="posts">
    {% for post in posts %}
    <article>
        <h2><a href="{{ post.get_absolute_url }}">{{ post.title }}</a></h2>
        <p>{{ post.excerpt }}</p>
    </article>
    {% endfor %}
</div>

{% include "pagination.html" %}
{% endblock %}

{% block extra_css %}
<style>
    .posts { margin: 20px; }
</style>
{% endblock %}
```

## 5.3 自定义标签和过滤器

### 自定义过滤器

```python
# templatetags/blog_extras.py
from django import template

register = template.Library()

@register.filter
def markdownify(text):
    import markdown
    return markdown.markdown(text)

@register.filter
def reading_time(text):
    words = len(text.split())
    minutes = words / 200
    return f"{int(minutes)} 分钟"
```

使用：

```html
{% load blog_extras %}

{{ post.content|markdownify }}
<p>阅读时间：{{ post.content|reading_time }}</p>
```

### 自定义标签

```python
# templatetags/blog_extras.py

@register.simple_tag
def get_recent_posts(count=5):
    from blog.models import Post
    return Post.objects.filter(
        status='published'
    ).order_by('-published_at')[:count]

@register.inclusion_tag('blog/recent_posts.html')
def show_recent_posts(count=5):
    posts = get_recent_posts(count)
    return {'posts': posts}
```

使用：

```html
<!-- 简单标签 -->
{% get_recent_posts 10 as recent_posts %}
{% for post in recent_posts %}
    <li>{{ post.title }}</li>
{% endfor %}

<!-- 包含标签 -->
{% show_recent_posts 5 %}
```

## 5.4 静态文件管理

### 配置

```python
# settings.py

STATIC_URL = '/static/'

STATICFILES_DIRS = [
    BASE_DIR / 'static',
]

STATIC_ROOT = BASE_DIR / 'staticfiles'

STATICFILES_FINDERS = [
    'django.contrib.staticfiles.finders.FileSystemFinder',
    'django.contrib.staticfiles.finders.AppDirectoriesFinder',
]
```

### 目录结构

```
project/
├── static/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── main.js
│   └── images/
│       └── logo.png
└── blog/
    └── static/
        └── blog/
            ├── css/
            └── js/
```

### 收集静态文件

```bash
# 开发环境
python manage.py runserver  # 自动提供静态文件

# 生产环境
python manage.py collectstatic  # 收集到 STATIC_ROOT
```

## 5.5 媒体文件

### 配置

```python
# settings.py

MEDIA_URL = '/media/'
MEDIA_ROOT = BASE_DIR / 'media'
```

### URL 配置

```python
# urls.py
from django.conf import settings
from django.conf.urls.static import static

urlpatterns = [
    # ...
]

if settings.DEBUG:
    urlpatterns += static(
        settings.MEDIA_URL,
        document_root=settings.MEDIA_ROOT
    )
```

### 模型中使用

```python
# models.py

class Post(models.Model):
    title = models.CharField(max_length=200)
    featured_image = models.ImageField(
        upload_to='posts/%Y/%m/',
        blank=True
    )
    
    def image_url(self):
        if self.featured_image:
            return self.featured_image.url
        return '/static/images/default.jpg'
```

### 模板中显示

```html
{% if post.featured_image %}
    <img src="{{ post.featured_image.url }}" alt="{{ post.title }}">
{% endif %}
```

## 5.6 表单渲染

### 基本表单

```html
<form method="POST">
    {% csrf_token %}
    
    {{ form.as_p }}
    
    <button type="submit">提交</button>
</form>
```

### 自定义渲染

```html
<form method="POST">
    {% csrf_token %}
    
    {% for field in form %}
    <div class="form-group">
        <label for="{{ field.id_for_label }}">{{ field.label }}</label>
        {{ field }}
        
        {% if field.errors %}
        <div class="errors">
            {% for error in field.errors %}
            <p>{{ error }}</p>
            {% endfor %}
        </div>
        {% endif %}
    </div>
    {% endfor %}
    
    <button type="submit">提交</button>
</form>
```
