# 第七章：函数

函数是组织代码、实现复用和模块化的核心机制。本章介绍 Python 函数的定义、参数、返回值及作用域规则。

------

## 7.1 定义与调用函数（`def`）

使用 `def` 关键字定义函数。

### 基本语法

```python
def function_name(parameters):
    """文档字符串（可选）"""
    statements
    return value  # 可选
```

### 示例

```python
def greet(name):
    """向指定用户打招呼"""
    print(f"Hello, {name}!")

# 调用函数
greet("Alice")  # 输出: Hello, Alice!
```

> 规范：
>
> - 函数名使用 `snake_case`
> - 括号后不加空格：`def func():`（不是 `def func ():`）

------

## 7.2 参数传递：位置参数、关键字参数

### 位置参数（Positional Arguments）

按定义顺序传入参数。

```python
def power(base, exponent):
    return base ** exponent

print(power(2, 3))  # 8（base=2, exponent=3）
```

### 关键字参数（Keyword Arguments）

通过参数名传递，顺序无关。

```python
print(power(exponent=3, base=2))  # 8
```

### 混合使用

- **位置参数必须在关键字参数之前**
- 同一参数不能重复赋值

```python
power(2, exponent=3)    # 合法
power(base=2, 3)        # SyntaxError!
power(2, base=2)        # TypeError!（base 被赋值两次）
```

## 7.3 返回值（`return`）与无返回值函数

### 有返回值

```python
def add(a, b):
    return a + b

result = add(3, 5)  # result = 8
```

### 无显式返回值

- 若函数无 `return` 或 `return` 不带值，则返回 `None`

```python
def say_hello():
    print("Hello!")

x = say_hello()  # 打印 "Hello!"，x = None
```

### 返回多个值

Python 支持返回元组（自动解包）：

```python
def min_max(numbers):
    return min(numbers), max(numbers)

nums = [1, 5, 3, 9, 2]
minimum, maximum = min_max(nums)
print(minimum, maximum)  # 1 9
```

## 7.4 默认参数与可变参数

### 默认参数（Default Arguments）

为参数提供默认值，调用时可省略。

```python
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")          # Hello, Alice!
greet("Bob", "Hi")      # Hi, Bob!
```

> 注意：**默认参数必须放在非默认参数之后**

### 可变位置参数（`*args`）

接收任意数量的位置参数，作为元组。

```python
def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3))      # 6
print(sum_all(10, 20, 30, 40))  # 100
```

### 可变关键字参数（`**kwargs`）

接收任意数量的关键字参数，作为字典。

```python
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30, city="Beijing")
# 输出:
# name: Alice
# age: 30
# city: Beijing
```

### 组合使用（顺序固定）

```python
def example(pos1, pos2, *args, kw1=None, **kwargs):
    pass

# 调用示例
example(1, 2, 3, 4, kw1="hello", extra="world")
```

参数顺序规则：
`位置参数 → *args → 关键字参数（含默认值） → **kwargs`

## 7.5 作用域与局部/全局变量

### 作用域规则（LEGB）

Python 按以下顺序查找变量：

1. **L**ocal（局部）— 函数内部
2. **E**nclosing（嵌套）— 外层函数
3. **G**lobal（全局）— 模块级别
4. **B**uilt-in（内置）— 如 `len`, `print`

### 局部变量 vs 全局变量

```python
x = 10  # 全局变量

def func():
    y = 5   # 局部变量
    print(x)  # 可读取全局变量
    print(y)

func()
# print(y)  # NameError! y 仅在 func 内有效
```

### 修改全局变量（`global`）

若要在函数内修改全局变量，需声明 `global`：

```python
counter = 0

def increment():
    global counter  # 声明使用全局变量
    counter += 1

increment()
print(counter)  # 1
```

> 警告：**避免过度使用 `global`**，易导致代码难以维护。

## 7.6 文档字符串（docstring）与函数注解（简介）

### 文档字符串（docstring）

函数第一行的字符串，用于描述功能，可通过 `help()` 查看。

```python
def calculate_area(length, width):
    """
    计算矩形面积。
    
    参数:
        length (float): 长度
        width (float): 宽度
    
    返回:
        float: 面积
    """
    return length * width

# 查看文档
help(calculate_area)
```

> 规范：使用三重双引号 `"""`，首行简述，空一行后详细说明。

### 函数注解（Type Hints，简介）

Python 3.5+ 支持类型提示，**不影响运行**，但提升可读性并支持 IDE 检查。

```python
def greet(name: str, age: int) -> str:
    return f"Hello, {name}! You are {age} years old."

# 注解存储在 __annotations__ 中
print(greet.__annotations__)
# {'name': <class 'str'>, 'age': <class 'int'>, 'return': <class 'str'>}
```

> 工具推荐：使用 `mypy` 进行静态类型检查。
