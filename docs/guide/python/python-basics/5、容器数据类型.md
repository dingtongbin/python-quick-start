# 第五章：容器数据类型

Python 提供了四种内置的容器类型，用于组织和操作多个数据项。本章详细介绍列表、元组、字典和集合的特性与使用方法。

------

## 5.1 列表（`list`）：可变序列

列表是**有序、可变、允许重复**的元素集合，用方括号 `[]` 表示。

### 创建列表

```python
# 空列表
empty_list = []

# 带初始值
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]  # 允许不同类型
```

### 索引与切片

- **索引**：从 0 开始，负数表示从末尾计数
- **切片**：`[start:stop:step]`，返回新列表

```python
colors = ["red", "green", "blue", "yellow"]

# 索引
print(colors[0])    # "red"
print(colors[-1])   # "yellow"

# 切片
print(colors[1:3])  # ["green", "blue"]（不包含索引 3）
print(colors[:2])   # ["red", "green"]
print(colors[::2])  # ["red", "blue"]（步长为 2）
```

### 常用方法

| 方法           | 说明                                    | 示例                        |
| -------------- | --------------------------------------- | --------------------------- |
| `append(x)`    | 在末尾添加元素                          | `fruits.append("orange")`   |
| `insert(i, x)` | 在索引 i 处左边插入                     | `fruits.insert(1, "mango")` |
| `remove(x)`    | 删除第一个匹配的 x                      | `fruits.remove("banana")`   |
| `pop([i])`     | 删除并返回索引 i 的元素（默认最后一个） | `last = fruits.pop()`       |
| `index(x)`     | 返回 x 的首次出现索引                   | `i = fruits.index("apple")` |
| `count(x)`     | 统计 x 出现次数                         | `n = fruits.count("apple")` |
| `sort()`       | 原地排序（升序）                        | `numbers.sort()`            |
| `reverse()`    | 原地反转                                | `fruits.reverse()`          |
| `clear()`      | 清空列表                                | `fruits.clear()`            |

> 注意：`sort()` 和 `reverse()` **修改原列表**，不返回新列表。

### 列表推导式（List Comprehension）

简洁创建列表的方式：

```python
# 生成平方数列表
squares = [x**2 for x in range(1, 6)]  # [1, 4, 9, 16, 25]

# 带条件
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]
```

------

## 5.2 元组（`tuple`）：不可变序列

元组是**有序、不可变、允许重复**的元素集合，用圆括号 `()` 表示。

### 创建元组

```python
# 空元组
empty_tuple = ()

# 单元素元组（注意逗号！）
single = (42,)  # 若无逗号，(42) 是整数

# 多元素
point = (3, 4)
person = ("Alice", 30, "Engineer")
```

### 特性

- **不可变**：创建后不能修改、添加或删除元素
- **性能略优于列表**（因结构固定）
- **可作为字典的键**（列表不行）

```python
coords = (10, 20)
# coords[0] = 5  # TypeError: 'tuple' object does not support item assignment
```

### 元组解包（Unpacking）

将元组元素赋值给多个变量：

```python
x, y = (1, 2)
name, age, job = ("Bob", 25, "Designer")

# 交换变量
a, b = 1, 2
a, b = b, a  # 无需临时变量
```

### 用途建议

- 表示固定结构的数据（如坐标、RGB 值）
- 函数返回多个值
- 保护数据不被意外修改

------

## 5.3 字典（`dict`）：键值对映射

字典是**无序（Python 3.7+ 保持插入顺序）、可变、键唯一**的键值对集合，用花括号 `{}` 表示。

### 创建字典

```python
# 空字典
empty_dict = {}

# 带初始值
student = {
    "name": "Alice",
    "age": 20,
    "courses": ["Math", "Physics"]
}

# 使用 dict() 构造器
person = dict(name="Bob", age=25)
```

### 访问与修改

```python
# 获取值
print(student["name"])        # "Alice"

# 安全获取（避免 KeyError）
print(student.get("grade", "N/A"))  # "N/A"（键不存在时返回默认值）

# 添加/修改
student["grade"] = "A"
student["age"] = 21

# 删除
del student["courses"]
```

### 常用方法

| 方法                | 说明                     | 示例                                  |
| ------------------- | ------------------------ | ------------------------------------- |
| `keys()`            | 返回所有键的视图         | `for k in student.keys(): ...`        |
| `values()`          | 返回所有值的视图         | `for v in student.values(): ...`      |
| `items()`           | 返回 (键, 值) 元组的视图 | `for k, v in student.items(): ...`    |
| `get(key, default)` | 安全获取值               | `student.get("email", "unknown")`     |
| `pop(key)`          | 删除并返回键对应的值     | `age = student.pop("age")`            |
| `update(other)`     | 合并另一个字典           | `student.update({"city": "Beijing"})` |

> 提示：`keys()`, `values()`, `items()` 返回的是**动态视图**，随字典变化而更新。

### 字典推导式

```python
# 平方映射
squares = {x: x**2 for x in range(1, 6)}
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

## 5.4 集合（`set`）：无序唯一元素

集合是**无序、可变、元素唯一**的集合，用花括号 `{}` 或 `set()` 创建。

### 创建集合

```python
# 空集合（必须用 set()）
empty_set = set()

# 非空集合
primes = {2, 3, 5, 7, 11}
letters = set("hello")  # {'h', 'e', 'l', 'o'}（自动去重）
```

> 注意：`{}` 创建的是空字典，不是空集合！

### 集合运算

| 运算     | 操作符 | 方法                     | 说明                     |
| -------- | ------ | ------------------------ | ------------------------ |
| 并集     | `      | `                        | `union()`                |
| 交集     | `&`    | `intersection()`         | 共同元素                 |
| 差集     | `-`    | `difference()`           | A 有但 B 没有            |
| 对称差集 | `^`    | `symmetric_difference()` | 不同时在 A 和 B 中的元素 |

```python
A = {1, 2, 3}
B = {3, 4, 5}

print(A | B)  # {1, 2, 3, 4, 5}
print(A & B)  # {3}
print(A - B)  # {1, 2}
print(A ^ B)  # {1, 2, 4, 5}
```

### 常用方法

- `add(x)`：添加元素
- `remove(x)`：删除元素（不存在则报错）
- `discard(x)`：删除元素（不存在不报错）
- `pop()`：随机弹出一个元素
- `clear()`：清空集合

### 主要用途

- **去重**：`unique = list(set(duplicates))`
- **成员测试**：`if x in my_set:`（比列表快得多）
- **数学集合运算**

## 5.5 容器嵌套与选择建议

### 嵌套示例

容器可以相互嵌套，构建复杂数据结构：

```python
# 列表的字典
students = [
    {"name": "Alice", "scores": [90, 85, 88]},
    {"name": "Bob", "scores": [78, 82, 80]}
]

# 字典的字典
config = {
    "database": {
        "host": "localhost",
        "port": 5432
    },
    "cache": {
        "enabled": True,
        "ttl": 300
    }
}

# 集合的元组（用于记录唯一组合）
visited = {(0, 0), (1, 0), (1, 1)}
```

### 如何选择容器？

| 需求                     | 推荐容器 | 理由                             |
| ------------------------ | -------- | -------------------------------- |
| 有序、可变、允许重复     | `list`   | 支持索引、切片、动态增删         |
| 有序、不可变、允许重复   | `tuple`  | 数据安全，可用作字典键           |
| 键值映射、快速查找       | `dict`   | O(1) 平均查找时间                |
| 唯一元素、去重、集合运算 | `set`    | 高效成员测试与数学运算           |
| 固定数量的异构数据       | `tuple`  | 如坐标 `(x, y)`、RGB `(r, g, b)` |

> 通用原则：
>
> - 默认使用 `list` 存储序列数据
> - 需要快速查找/映射时用 `dict`
> - 需要去重或做集合运算时用 `set`
> - 数据不应被修改时用 `tuple`
