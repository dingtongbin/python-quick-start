# 第九章：文件操作

文件操作是程序与外部存储交互的基础。本章介绍如何安全地读写文件、处理路径及管理资源。

------

## 9.1 打开与关闭文件（`open()`, `with` 语句）

### `open()` 函数

使用 `open()` 打开文件，返回**文件对象**。

#### 基本语法

```python
file = open(filename, mode, encoding)
```

#### 常用模式（`mode`）

| 模式  | 说明                            |
| ----- | ------------------------------- |
| `'r'` | 只读（默认）                    |
| `'w'` | 写入（覆盖原内容）              |
| `'a'` | 追加（在末尾写入）              |
| `'x'` | 独占创建（文件存在则失败）      |
| `'b'` | 二进制模式（如 `'rb'`, `'wb'`） |
| `'t'` | 文本模式（默认，可省略）        |

> 必须指定 `encoding='utf-8'`（文本模式），避免跨平台乱码。

### 手动关闭（不推荐）

```python
f = open('data.txt', 'r', encoding='utf-8')
content = f.read()
f.close()  # 必须显式关闭
```

>  风险：若中间发生异常，`close()` 可能不会执行，导致资源泄漏。

### 使用 `with` 语句（推荐）

自动管理资源，确保文件在使用后关闭。

```python
with open('data.txt', 'r', encoding='utf-8') as f:
    content = f.read()
# 文件在此处自动关闭，即使发生异常
```

> 最佳实践：**始终使用 `with` 语句操作文件**

## 9.2 读取文件：`read()`, `readline()`, `readlines()`

### `read(size)`

- 读取指定字节数（文本模式为字符数）
- 若省略 `size`，读取整个文件

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    data = f.read(100)  # 读取前100个字符
    all_data = f.read()  # 读取剩余全部内容
```

### `readline()`

- 读取一行（包含换行符 `\n`）
- 返回空字符串表示文件结束

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    line1 = f.readline()  # 第一行
    line2 = f.readline()  # 第二行
```

### `readlines()`

- 读取所有行，返回**列表**（每行含 `\n`）

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()
    for line in lines:
        print(line.strip())  # 去除换行符
```

### 逐行迭代（内存高效）

对大文件推荐直接迭代文件对象：

```python
with open('large_file.txt', 'r', encoding='utf-8') as f:
    for line in f:  # 逐行读取，不一次性加载到内存
        process(line)
```

## 9.3 写入文件：`write()`, `writelines()`

### `write(string)`

- 写入字符串（文本模式）或字节（二进制模式）
- **不自动添加换行符**

```python
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write("Hello\n")
    f.write("World\n")
```

### `writelines(lines)`

- 写入字符串序列（如列表）
- **不自动添加换行符**，需自行处理

```python
lines = ["Line 1\n", "Line 2\n"]
with open('output.txt', 'w', encoding='utf-8') as f:
    f.writelines(lines)
```

> 注意：`'w'` 模式会**清空原文件**！追加请用 `'a'` 模式。

## 9.4 文件路径处理（`os.path` 与 `pathlib` 简介）

### 传统方式：`os.path`

```python
import os

# 拼接路径（跨平台）
path = os.path.join('folder', 'subfolder', 'file.txt')

# 获取绝对路径
abs_path = os.path.abspath('file.txt')

# 检查文件是否存在
if os.path.exists('data.txt'):
    print("文件存在")

# 分离目录与文件名
dirname = os.path.dirname(path)
basename = os.path.basename(path)
```

### 现代方式：`pathlib`（Python 3.4+ 推荐）

面向对象的路径操作：

```python
from pathlib import Path

# 创建路径对象
p = Path('folder') / 'subfolder' / 'file.txt'

# 常用操作
print(p.absolute())          # 绝对路径
print(p.exists())            # 是否存在
print(p.parent)              # 父目录
print(p.name)                # 文件名（含扩展名）
print(p.stem)                # 文件名（不含扩展名）
print(p.suffix)              # 扩展名（如 '.txt'）

# 读写文件（简化操作）
p.write_text("Hello, World!", encoding='utf-8')
content = p.read_text(encoding='utf-8')
```

> 推荐：**新项目优先使用 `pathlib.Path`**，代码更简洁、可读性更强。

## 9.5 异常安全的文件操作

文件操作可能因权限、路径不存在等原因失败，需捕获异常。

### 常见异常

- `FileNotFoundError`：文件或目录不存在
- `PermissionError`：无访问权限
- `IsADirectoryError`：尝试对目录执行文件操作

### 安全示例

```python
from pathlib import Path

def safe_read_file(filepath):
    """
    安全读取文件内容。
    
    参数:
        filepath (str 或 Path): 文件路径
    
    返回:
        str: 文件内容，或 None（若失败）
    """
    try:
        path = Path(filepath)
        if not path.is_file():
            print(f"错误: {filepath} 不是一个文件")
            return None
        
        return path.read_text(encoding='utf-8')
    
    except FileNotFoundError:
        print(f"错误: 文件 {filepath} 不存在")
    except PermissionError:
        print(f"错误: 无权访问 {filepath}")
    except UnicodeDecodeError:
        print(f"错误: 文件 {filepath} 编码不是 UTF-8")
    except Exception as e:
        print(f"未知错误: {e}")
    
    return None

# 使用
content = safe_read_file("config.txt")
if content is not None:
    print(content)
```

### 写入时的安全检查

```python
def safe_write_file(filepath, content):
    """安全写入文件，确保父目录存在"""
    try:
        path = Path(filepath)
        path.parent.mkdir(parents=True, exist_ok=True)  # 自动创建目录
        path.write_text(content, encoding='utf-8')
        print(f"成功写入 {filepath}")
    except Exception as e:
        print(f"写入失败: {e}")
```

> 关键原则：
>
> - **先检查再操作**（如 `path.exists()`）
> - **捕获具体异常**，而非笼统的 `except:`
> - **使用 `pathlib` 简化路径与文件操作**
