## 12.1 项目：简易通讯录管理系统

该系统允许用户通过命令行界面管理联系人信息，支持添加、查询、删除操作，并将数据保存到本地 JSON 文件中，实现持久化。

------

## 12.2 功能要求

| 功能         | 描述                                                    |
| ------------ | ------------------------------------------------------- |
| 添加联系人   | 输入姓名和电话号码，存入内存并写入文件                  |
| 查询联系人   | 按姓名模糊或精确查找，返回匹配结果                      |
| 删除联系人   | 按姓名删除（若存在多个同名，全部删除）                  |
| 数据持久化   | 使用 JSON 格式将联系人列表保存到 `contacts.json`        |
| 用户交互菜单 | 循环显示菜单，接收用户输入，执行对应操作，输入 `0` 退出 |

------

## 12.3 代码组织与调试技巧

### 代码结构说明

- 使用 `Contact` 类封装单个联系人信息。
- 使用 `AddressBook` 类管理联系人集合及文件读写。
- 主程序负责用户交互逻辑。
- 所有文件操作使用 `try...except` 捕获 I/O 异常。
- 使用 `json` 模块进行序列化/反序列化。

## 核心类设计

### `Contact` 类：表示单个联系人

| 功能简述   | 调用方式               | 说明                                                     |
| ---------- | ---------------------- | -------------------------------------------------------- |
| 创建联系人 | `Contact(name, phone)` | 必须参数：`name`（str）、`phone`（str）；自动校验非空    |
| 获取姓名   | `contact.name`         | 公有属性，只读（通过构造初始化）                         |
| 获取电话   | `contact.phone`        | 公有属性                                                 |
| 转为字典   | `contact.to_dict()`    | 返回 `{"name": "...", "phone": "..."}`，用于 JSON 序列化 |

### `AddressBook` 类：管理联系人集合

| 功能简述           | 调用方式                                | 说明                                                         |
| ------------------ | --------------------------------------- | ------------------------------------------------------------ |
| 初始化通讯录       | `AddressBook(filename="contacts.json")` | 可选参数：文件路径，默认 `contacts.json`；自动尝试加载已有数据 |
| 添加联系人         | `add_contact(contact)`                  | 参数：`Contact` 实例；自动去重（同名同号视为重复）           |
| 查询联系人（模糊） | `search(name)`                          | 参数：部分或完整姓名；返回匹配的 `Contact` 列表              |
| 删除联系人         | `remove(name)`                          | 删除所有姓名匹配的联系人；返回删除数量                       |
| 保存到文件         | `save()`                                | 将当前联系人列表写入 JSON 文件                               |
| 加载文件           | `load()`                                | 从 JSON 文件读取数据并覆盖当前列表；若文件不存在则初始化为空列表 |

## 完整代码示例（含注释与错误处理）

```python
import json
import os
from typing import List


class Contact:
    """表示一个联系人"""
    def __init__(self, name: str, phone: str):
        if not name or not phone:
            raise ValueError("姓名和电话不能为空")
        self.name = name.strip()
        self.phone = phone.strip()

    def to_dict(self) -> dict:
        """转换为字典，便于序列化"""
        return {"name": self.name, "phone": self.phone}

    def __repr__(self) -> str:
        return f"Contact(name='{self.name}', phone='{self.phone}')"


class AddressBook:
    """通讯录管理类"""
    def __init__(self, filename: str = "contacts.json"):
        self.filename = filename
        self.contacts: List[Contact] = []
        self.load()  # 启动时尝试加载已有数据

    def load(self) -> None:
        """从 JSON 文件加载联系人"""
        if not os.path.exists(self.filename):
            self.contacts = []
            return

        try:
            with open(self.filename, 'r', encoding='utf-8') as f:
                data = json.load(f)
                self.contacts = [Contact(item['name'], item['phone']) for item in data]
        except (json.JSONDecodeError, KeyError, IOError) as e:
            print(f"警告：加载 {self.filename} 失败，使用空通讯录。错误: {e}")
            self.contacts = []

    def save(self) -> None:
        """将联系人保存到 JSON 文件"""
        try:
            with open(self.filename, 'w', encoding='utf-8') as f:
                json.dump(
                    [contact.to_dict() for contact in self.contacts],
                    f,
                    ensure_ascii=False,
                    indent=2
                )
        except IOError as e:
            print(f"错误：无法保存到 {self.filename}。{e}")

    def add_contact(self, contact: Contact) -> bool:
        """添加联系人，若已存在（同名同号）则不添加"""
        for c in self.contacts:
            if c.name == contact.name and c.phone == contact.phone:
                return False  # 已存在
        self.contacts.append(contact)
        self.save()
        return True

    def search(self, name: str) -> List[Contact]:
        """按姓名模糊查询（不区分大小写）"""
        name_lower = name.lower().strip()
        return [c for c in self.contacts if name_lower in c.name.lower()]

    def remove(self, name: str) -> int:
        """删除所有姓名匹配的联系人，返回删除数量"""
        original_count = len(self.contacts)
        self.contacts = [c for c in self.contacts if c.name != name]
        deleted = original_count - len(self.contacts)
        if deleted > 0:
            self.save()
        return deleted


def main():
    """主程序：用户交互菜单"""
    book = AddressBook()

    while True:
        print("\n=== 简易通讯录管理系统 ===")
        print("1. 添加联系人")
        print("2. 查询联系人")
        print("3. 删除联系人")
        print("0. 退出")
        choice = input("请选择操作 (0-3): ").strip()

        if choice == "0":
            print("再见！")
            break

        elif choice == "1":
            try:
                name = input("请输入姓名: ").strip()
                phone = input("请输入电话: ").strip()
                contact = Contact(name, phone)
                if book.add_contact(contact):
                    print("联系人添加成功！")
                else:
                    print("联系人已存在，未重复添加。")
            except ValueError as e:
                print(f"输入错误: {e}")

        elif choice == "2":
            name = input("请输入要查询的姓名: ").strip()
            if not name:
                print("姓名不能为空。")
                continue
            results = book.search(name)
            if results:
                print(f"\n找到 {len(results)} 个联系人:")
                for i, c in enumerate(results, 1):
                    print(f"{i}. {c.name} - {c.phone}")
            else:
                print("未找到匹配的联系人。")

        elif choice == "3":
            name = input("请输入要删除的姓名: ").strip()
            if not name:
                print("姓名不能为空。")
                continue
            deleted = book.remove(name)
            if deleted > 0:
                print(f"成功删除 {deleted} 个联系人。")
            else:
                print("未找到该姓名的联系人。")

        else:
            print("无效选项，请重新输入。")


if __name__ == "__main__":
    main()
```

## 整体调用注意事项

### 关于返回值

- `add_contact()` 返回 `bool`：`True` 表示成功添加，`False` 表示已存在。
- `remove()` 返回 `int`：表示实际删除的联系人数量。
- `search()` 返回 `List[Contact]`：可能为空列表，调用方需判断。
- 所有文件操作（`load`, `save`）在失败时打印警告但不抛出异常，保证程序继续运行。

### 异常处理原则

- **输入校验**：在 `Contact` 构造时校验非空，避免无效数据。
- **文件安全**：使用 `try...except` 包裹文件读写，防止程序崩溃。
- **用户友好**：捕获 `ValueError` 并提示用户重新输入。

### 调试技巧

1. **临时打印状态**：在关键方法（如 `save`、`load`）中加入 `print` 调试信息（正式发布前移除）。
2. **单元测试**：可为 `Contact` 和 `AddressBook` 编写独立测试函数，验证核心逻辑。
3. **检查 JSON 文件**：手动查看 `contacts.json` 内容，确认数据格式正确。
4. **边界测试**：尝试输入空字符串、特殊字符、超长姓名等，验证鲁棒性。
