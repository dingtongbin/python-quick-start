# 第四章：控制流程

程序的执行通常不是线性的。控制流程语句允许程序根据条件或重复逻辑改变执行路径。本章介绍 Python 中的条件判断和循环结构。

------

## 4.1 条件语句：`if` / `elif` / `else`

条件语句根据布尔表达式的值决定执行哪段代码。

### 基本语法

```python
if condition1:
    # condition1 为 True 时执行
    statement1
elif condition2:
    # condition1 为 False 且 condition2 为 True 时执行
    statement2
else:
    # 所有条件均为 False 时执行
    statement3
```

### 示例

```python
age = 20

if age < 13:
    print("儿童")
elif age < 20:
    print("青少年")
elif age < 65:
    print("成年人")
else:
    print("老年人")
```

> 注意：
>
> - `elif` 和 `else` 是可选的
> - 可以有多个 `elif`，但最多一个 `else`
> - **缩进是语法的一部分**（通常 4 个空格）

### 真值测试（Truthy/Falsy）

Python 会自动将非布尔值转换为布尔值进行判断：

- **Falsy 值**：`None`, `False`, `0`, `0.0`, `""`, `[]`, `{}`, `set()`
- **其他值均为 Truthy**

```python
name = ""
if name:  # 等价于 if len(name) > 0:
    print("有名字")
else:
    print("名字为空")  # 输出此行
```

## 4.2 循环语句

### `while` 循环

当条件为 `True` 时重复执行代码块。

#### 语法

```python
while condition:
    statement
```

#### 示例：计算阶乘

```python
n = 5
factorial = 1
i = 1

while i <= n:
    factorial *= i
    i += 1

print(f"{n}! = {factorial}")  # 5! = 120
```

> 警告：确保循环条件最终会变为 `False`，否则会导致**无限循环**。

### `for` 循环（遍历序列）

用于遍历任何可迭代对象（如字符串、列表、元组、字典等）。

#### 语法

```python
for item in iterable:
    statement
```

#### 示例

```python
# 遍历列表
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 遍历字符串
for char in "Hello":
    print(char)

# 遍历字典（默认遍历键）
person = {"name": "Alice", "age": 30}
for key in person:
    print(f"{key}: {person[key]}")
```

> 提示：`for` 循环在 Python 中是“for-each”风格，不直接暴露索引。

## 4.3 循环控制：`break` 与 `continue`

### `break`

立即终止整个循环，跳出循环体。

```python
# 查找第一个负数
numbers = [1, 3, -2, 4, 5]
for num in numbers:
    if num < 0:
        print(f"找到负数: {num}")
        break
```

### `continue`

跳过当前迭代的剩余部分，直接进入下一次循环。

```python
# 打印奇数
for i in range(1, 11):
    if i % 2 == 0:
        continue
    print(i)  # 输出 1, 3, 5, 7, 9
```

### `else` 子句（用于循环）

- `while`/`for` 循环可带 `else` 子句
- **仅当循环正常结束（未被 `break` 中断）时执行**

```python
# 检查是否包含偶数
numbers = [1, 3, 5, 7]
for num in numbers:
    if num % 2 == 0:
        print("找到偶数")
        break
else:
    print("未找到偶数")  # 输出此行
```

## 4.4 `range()` 函数的使用

`range()` 生成一个不可变的整数序列，常用于 `for` 循环中模拟传统计数循环。

### 语法

```python
range(stop)           # 从 0 到 stop-1
range(start, stop)    # 从 start 到 stop-1
range(start, stop, step)  # 指定步长
```

### 示例

```python
# 基本用法
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# 指定起始值
for i in range(2, 6):
    print(i)  # 2, 3, 4, 5

# 指定步长（包括负数）
for i in range(10, 0, -2):
    print(i)  # 10, 8, 6, 4, 2

# 转换为列表（调试用）
print(list(range(0, 10, 2)))  # [0, 2, 4, 6, 8]
```

> 注意：
>
> - `range()` 返回的是 `range` 对象（惰性求值），不是列表
> - 不支持浮点数参数

## 4.5 综合示例：猜数字游戏

结合本章所学，实现一个简单的猜数字游戏。

```python
import random

def guess_number_game():
    """
    猜数字游戏：
    - 程序随机生成 1~100 的整数
    - 用户输入猜测
    - 程序提示“太大”或“太小”
    - 猜中后显示尝试次数
    """
    secret_number = random.randint(1, 100)
    attempts = 0
    print("我想了一个 1 到 100 之间的数字，你能猜中吗？")

    while True:
        try:
            # 获取用户输入
            guess = int(input("请输入你的猜测: "))
            attempts += 1

            # 判断结果
            if guess < secret_number:
                print("太小了！")
            elif guess > secret_number:
                print("太大了！")
            else:
                print(f"恭喜你！猜对了！数字是 {secret_number}")
                print(f"你总共尝试了 {attempts} 次。")
                break  # 游戏结束

        except ValueError:
            print("请输入一个有效的整数！")

# 启动游戏
if __name__ == "__main__":
    guess_number_game()
```

### 代码要点说明

- 使用 `while True` + `break` 实现主循环
- `random.randint(1, 100)` 生成随机整数
- `try...except` 处理无效输入（后续章节会深入）
- 通过 `elif` 实现三路分支判断

> 运行效果：
>
> ```
> 我想了一个 1 到 100 之间的数字，你能猜中吗？
> 请输入你的猜测: 50
> 太大了！
> 请输入你的猜测: 25
> 太小了！
> 请输入你的猜测: 37
> 恭喜你！猜对了！数字是 37
> 你总共尝试了 3 次。
> ```
