# 🐍 Python 入门：从零开始学编程

欢迎来到 Python 的世界！本教程专为 **零基础** 学习者设计，不需要任何编程经验。

所有示例基于 **Python 3.10**，循序渐进，强调动手实践。学完这 13 章，你将具备独立编写 Python 程序的能力。

---

## 💡 学习建议

> **不要只看，要动手敲代码。** 每个示例都自己运行一遍，改改参数看看会怎样。这是学编程最快的方式。

- ⏱️ 预计学习时间：**2-4 周**（每天 1-2 小时）
- 🎯 学完能做什么：自动化脚本、数据处理、命令行工具
- 📌 后续方向：[数据分析](/guide/python/python-data-analysis/) / [Web 开发](/guide/python/python-django/) / [爬虫](/guide/python/python-web-scraping/)

## ❓ 常见问题

**Q: 我完全没有编程基础，能学会吗？**
A: 可以。本教程就是为零基础设计的，从最简单的 `print` 开始，一步步来。

**Q: 需要安装什么软件？**
A: 只需要 Python 3.10 和一个代码编辑器（推荐 VS Code）。第 1 章会详细说明。

**Q: 学完基础后该学什么？**
A: 看你的目标——想做数据分析就学 Pandas，想做网站就学 Django，想找工作就刷算法。详见[推荐学习路线](/guide/python/python-basics/#推荐学习路线)。
