# 第六章：字符串操作

字符串是 Python 中最常用的数据类型之一。本章介绍字符串的格式化、常用方法、多行处理及编码基础。

------

## 6.1 字符串格式化

将变量嵌入字符串有多种方式，推荐使用 **f-string**（Python 3.6+）。

### f-string（推荐）

在字符串前加 `f` 或 `F`，用 `{}` 包裹表达式。

```python
name = "Alice"
age = 30
score = 95.7

# 基本用法
message = f"Hello, {name}! You are {age} years old."
print(message)  # Hello, Alice! You are 30 years old.

# 表达式与格式控制
print(f"Score: {score:.1f}")        # Score: 95.7（保留1位小数）
print(f"Hex: {255:#x}")             # Hex: 0xff
print(f"Item count: {len([1,2,3])}") # Item count: 3
```

> 优点：简洁、高效、支持任意表达式。

### `.format()` 方法

使用 `{}` 作为占位符，通过 `.format()` 填充。

```python
# 位置参数
msg1 = "Hello, {}! You are {} years old.".format(name, age)

# 索引参数
msg2 = "Hello, {0}! {0} is {1} years old.".format(name, age)

# 关键字参数
msg3 = "Hello, {n}! You are {a} years old.".format(n=name, a=age)

# 格式控制
msg4 = "Score: {:.1f}".format(score)
```

>  适用场景：需要兼容 Python 3.5 及以下版本时使用。

### `%` 格式化（了解即可）

源自 C 语言的旧式格式化，**不推荐新代码使用**。

```python
msg = "Hello, %s! You are %d years old." % (name, age)
score_msg = "Score: %.1f" % score
```

>  缺点：易出错、不支持命名参数、性能较差。

## 6.2 常用字符串方法

字符串是不可变对象，所有方法均返回新字符串。

| 方法                  | 说明                                  | 示例                                   |
| --------------------- | ------------------------------------- | -------------------------------------- |
| `split(sep)`          | 按分隔符分割为列表                    | `"a,b,c".split(",") → ["a", "b", "c"]` |
| `join(iterable)`      | 用字符串连接序列元素                  | `",".join(["a", "b"]) → "a,b"`         |
| `strip([chars])`      | 去除首尾空白（或指定字符）            | `" hello ".strip() → "hello"`          |
| `replace(old, new)`   | 替换子串                              | `"hello".replace("l", "L") → "heLLo"`  |
| `find(sub)`           | 返回子串首次出现索引（-1 表示未找到） | `"hello".find("e") → 1`                |
| `upper()` / `lower()` | 转换大小写                            | `"Hello".upper() → "HELLO"`            |
| `startswith(prefix)`  | 是否以某前缀开头                      | `"file.txt".startswith("file") → True` |
| `endswith(suffix)`    | 是否以某后缀结尾                      | `"image.png".endswith(".png") → True`  |

### 综合示例：清洗数据

```python
data = "  Alice, 30, Engineer  \n"

# 清洗并解析
cleaned = data.strip()                 # 去除首尾空白和换行
parts = cleaned.split(",")             # 分割
name = parts[0].strip()                # 去除字段内空白
age = int(parts[1].strip())
job = parts[2].strip()

print(f"Name: {name}, Age: {age}, Job: {job}")
# 输出: Name: Alice, Age: 30, Job: Engineer
```

## 6.3 多行字符串与转义字符

### 多行字符串

使用三重引号（`'''` 或 `"""`）定义跨多行的字符串。

```python
# 保留换行和缩进
sql = """
SELECT name, age
FROM users
WHERE age > 18;
"""

# 注意：首尾换行会被包含
text = """Line 1
Line 2"""
print(repr(text))  # 'Line 1\nLine 2'
```

> 技巧：若不想包含首行换行，可写成：
>
> ```python
> text = """\
> Line 1
> Line 2"""
> ```

### 转义字符

用于表示特殊字符：

| 转义序列 | 含义   |
| -------- | ------ |
| `\n`     | 换行   |
| `\t`     | 制表符 |
| `\\`     | 反斜杠 |
| `\"`     | 双引号 |
| `\'`     | 单引号 |

### 原始字符串（Raw String）

在字符串前加 `r`，禁用转义（常用于正则表达式、文件路径）。

```python
# 普通字符串
path = "C:\\Users\\Alice\\Documents"  # 需双反斜杠

# 原始字符串
path = r"C:\Users\Alice\Documents"    # 更清晰

# 正则表达式
pattern = r"\d{3}-\d{2}-\d{4}"  # 匹配社保号
```

> 注意：原始字符串不能以奇数个反斜杠结尾（如 `r"\"` 会报错）。

## 6.4 编码基础：UTF-8 与 `encode()`/`decode()`

### 字符串与字节

- **字符串（`str`）**：人类可读的文本（Unicode）
- **字节（`bytes`）**：计算机存储/传输的二进制数据

### 编码与解码

- **编码（encode）**：`str` → `bytes`
- **解码（decode）**：`bytes` → `str`

```python
text = "你好，世界！"

# 编码为 UTF-8 字节
encoded = text.encode("utf-8")
print(encoded)  # b'\xe4\xbd\xa0\xe5\xa5\xbd\xef\xbc\x8c\xe4\xb8\x96\xe7\x95\x8c\xef\xbc\x81'

# 解码回字符串
decoded = encoded.decode("utf-8")
print(decoded)  # "你好，世界！"
```

### 常见编码

- **UTF-8**：互联网标准，兼容 ASCII，推荐使用
- **GBK**：中文 Windows 系统常用（不推荐新项目使用）

### 文件读写中的编码

```python
# 写入 UTF-8 文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("你好")

# 读取 UTF-8 文件
with open("output.txt", "r", encoding="utf-8") as f:
    content = f.read()
```

> 重要：**始终显式指定 `encoding`**，避免系统默认编码导致乱码。
