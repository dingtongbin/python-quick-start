# 第三章：运算符与表达式

本章介绍 Python 中的各类运算符及其使用规则。运算符用于对数据进行操作，是构建表达式和实现逻辑的基础。

------

## 3.1 算术运算符（`+`, `-`, `*`, `/`, `//`, `%`, `**`）

算术运算符用于执行基本数学运算。

| 运算符 | 名称   | 示例     | 结果  | 说明                               |
| ------ | ------ | -------- | ----- | ---------------------------------- |
| `+`    | 加法   | `5 + 3`  | `8`   | 也可用于字符串拼接                 |
| `-`    | 减法   | `5 - 3`  | `2`   | —                                  |
| `*`    | 乘法   | `5 * 3`  | `15`  | 也可用于字符串重复                 |
| `/`    | 除法   | `7 / 2`  | `3.5` | **总是返回浮点数**                 |
| `//`   | 整除   | `7 // 2` | `3`   | 向负无穷方向取整（floor division） |
| `%`    | 取模   | `7 % 2`  | `1`   | 返回除法余数                       |
| `**`   | 幂运算 | `2 ** 3` | `8`   | 等价于 `pow(2, 3)`                 |

### 特殊用法示例

```python
# 字符串拼接
greeting = "Hello" + " " + "World"  # "Hello World"

# 字符串重复
line = "-" * 40  # "----------------------------------------"

# 负数整除（注意 floor 行为）
print(-7 // 2)   # -4（不是 -3！）
```

> 注意：`/` 在 Python 3 中**总是浮点除法**，即使操作数都是整数。

## 3.2 比较运算符（`==`, `!=`, `<`, `>`, `<=`, `>=`）

比较运算符用于比较两个值，返回布尔值 `True` 或 `False`。

| 运算符 | 名称     | 示例     | 结果   |
| ------ | -------- | -------- | ------ |
| `==`   | 等于     | `5 == 5` | `True` |
| `!=`   | 不等于   | `5 != 3` | `True` |
| `<`    | 小于     | `3 < 5`  | `True` |
| `>`    | 大于     | `5 > 3`  | `True` |
| `<=`   | 小于等于 | `5 <= 5` | `True` |
| `>=`   | 大于等于 | `5 >= 3` | `True` |

### 链式比较

Python 支持数学风格的链式比较：

```python
# 等价于 (1 < x) and (x < 10)
if 1 < x < 10:
    print("x 在 1 到 10 之间")
```

### 对象比较 vs 值比较

- `==` 比较**值是否相等**
- `is` 比较**是否为同一对象**（身份比较）

```python
a = [1, 2]
b = [1, 2]
print(a == b)  # True（值相同）
print(a is b)  # False（不同对象）

c = a
print(a is c)  # True（同一对象）
```

> 建议：**永远用 `==` 比较值，仅在需要检查对象身份时用 `is`**（如 `if x is None`）。

## 3.3 逻辑运算符（`and`, `or`, `not`）

逻辑运算符用于组合布尔表达式。

| 运算符 | 名称   | 示例               | 说明                             |
| ------ | ------ | ------------------ | -------------------------------- |
| `and`  | 逻辑与 | `x > 0 and x < 10` | 两边都为 `True` 时返回 `True`    |
| `or`   | 逻辑或 | `x < 0 or x > 100` | 任一边为 `True` 时返回 `True`    |
| `not`  | 逻辑非 | `not (x == 0)`     | 取反：`True` → `False`，反之亦然 |

### 短路求值（Short-circuit Evaluation）

- `and`：若左操作数为 `False`，不计算右操作数
- `or`：若左操作数为 `True`，不计算右操作数

```python
# 安全访问列表元素
items = []
if items and items[0] == "apple":  # 若 items 为空，不会访问 items[0]
    print("First item is apple")
```

### 逻辑运算的返回值

Python 的逻辑运算符**不强制返回布尔值**，而是返回实际参与运算的对象：

```python
print(0 and "hello")   # 0（因为 0 是 falsy，直接返回）
print("hi" or "world") # "hi"（因为 "hi" 是 truthy）
print(not "hello")     # False（not 总是返回布尔值）
```

## 3.4 赋值运算符（`=` 与复合赋值如 `+=`）

赋值运算符用于将值绑定到变量。

### 基本赋值

```python
x = 10
```

### 复合赋值（Augmented Assignment）

将运算与赋值结合，等价于 `x = x op y`。

| 运算符 | 示例      | 等价形式     |
| ------ | --------- | ------------ |
| `+=`   | `x += 5`  | `x = x + 5`  |
| `-=`   | `x -= 3`  | `x = x - 3`  |
| `*=`   | `x *= 2`  | `x = x * 2`  |
| `/=`   | `x /= 4`  | `x = x / 4`  |
| `//=`  | `x //= 3` | `x = x // 3` |
| `%=`   | `x %= 7`  | `x = x % 7`  |
| `**=`  | `x **= 2` | `x = x ** 2` |

### 多重赋值

```python
# 同时赋值多个变量
a, b = 1, 2

# 交换变量（无需临时变量）
a, b = b, a
```

> 💡 提示：复合赋值对可变对象有特殊行为（如列表的 `+=` 是原地修改，而 `=` 创建新对象）。

## 3.5 运算符优先级与括号使用

当表达式包含多个运算符时，Python 按**优先级**和**结合性**决定计算顺序。

### 优先级从高到低（部分）

1. 括号 `()`
2. 幂运算 `**`（右结合）
3. 正负号 `+x`, `-x`
4. 乘除模 `*`, `/`, `//`, `%`
5. 加减 `+`, `-`
6. 比较 `==`, `!=`, `<`, `>`, `<=`, `>=`
7. 逻辑非 `not`
8. 逻辑与 `and`
9. 逻辑或 `or`

### 示例

```python
# 无括号：按优先级计算
result = 2 + 3 * 4    # 14（先乘后加）

# 使用括号明确意图
result = (2 + 3) * 4  # 20

# 混合逻辑与比较
valid = age >= 18 and score > 90 or is_admin
# 等价于：
valid = ((age >= 18) and (score > 90)) or is_admin
```

> 最佳实践：**即使优先级明确，也建议用括号提高可读性**。
