# 第一章：环境准备与第一个程序

本章将引导你完成 Python 开发环境的搭建，并运行你的第一个 Python 程序。所有内容基于 **Python 3.10**，这是目前广泛使用且功能稳定的版本。

------

## 1.1 Python 简介与版本选择（为什么用 3.10）

Python 是一种高级、解释型、通用编程语言，以其简洁的语法和强大的标准库著称。它广泛应用于 Web 开发、数据分析、人工智能、自动化脚本等领域。

### 为什么选择 Python 3.10？

- **主流支持**：Python 3.10 于 2021 年 10 月发布，是长期受社区和企业支持的稳定版本。
- **兼容性好**：绝大多数第三方库（如 `requests`, `numpy`, `pandas`）均支持 Python 3.10。
- **教学友好**：语法清晰，适合初学者建立正确编程习惯。

>  注意：**不要使用 Python 2.x**，它已于 2020 年停止维护。

------

## 1.2 安装 Python（Windows / macOS / Linux）

### Windows

1. 访问 [Python 官网下载页](https://www.python.org/downloads/)

2. 下载 **Python 3.10.x** 的 Windows installer（推荐 64 位）

3. 运行安装程序，**务必勾选 “Add Python to PATH”**

4. 点击 “Install Now”

5. 验证安装：打开命令提示符（CMD），输入：

   ```bash
   python --version
   ```

   应显示 

   ```
   Python 3.10.x
   ```



### macOS

#### 方法一：使用官方安装包

1. 从 [Python 官网](https://www.python.org/downloads/macos/) 下载 macOS 64-bit installer

2. 双击 `.pkg` 文件，按提示安装

3. 打开终端，验证：

   ```bash
   python3 --version
   ```

#### 方法二：使用 Homebrew（推荐开发者）

```bash
# 安装 Homebrew（若未安装）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Python 3.10
brew install python@3.10

# 验证
python3.10 --version
```

### Linux（以 Ubuntu/Debian 为例）

大多数 Linux 发行版预装 Python，但可能不是 3.10。建议通过以下方式安装：

```bash
# 更新包列表
sudo apt update

# 安装 Python 3.10（Ubuntu 22.04+ 默认包含）
sudo apt install python3.10

# 安装 pip（包管理器）
sudo apt install python3.10-pip

# 验证
python3.10 --version
```

> 提示：为方便使用，可创建软链接：
>
> ```bash
> sudo ln -s /usr/bin/python3.10 /usr/local/bin/python
> ```

## 1.3 配置开发环境（IDLE、VS Code、PyCharm 简介）

编写 Python 代码需要一个**代码编辑器**或**集成开发环境（IDE）**。

### 选项一：IDLE（Python 自带）

- **优点**：无需额外安装，轻量级，适合初学者。

- 启动方式：

  - Windows：开始菜单 → Python 3.10 → IDLE
  - macOS/Linux：终端输入 `idle3`
  
- **功能**：基本语法高亮、交互式 Shell、简单调试。

### 选项二：Visual Studio Code（推荐）

- **优点**：免费、轻量、插件丰富、跨平台。

- 配置步骤：

  1. 下载并安装 [VS Code](https://code.visualstudio.com/)
  2. 安装扩展：**Python**（由 Microsoft 提供）
  3. 打开文件夹，新建 `.py` 文件，VS Code 会自动提示选择 Python 解释器
  
- **优势**：智能补全、调试器、Git 集成、Jupyter 支持。

### 选项三：PyCharm Community（免费版）

- **优点**：专为 Python 设计，功能强大。
- **适用场景**：中大型项目开发。
- **注意**：Community 版免费，Professional 版需付费（含 Web 开发支持）。

> 初学者建议：**VS Code + Python 扩展**，兼顾易用性与扩展性。

## 1.4 编写并运行第一个程序：`print("Hello, World!")`

### 步骤 1：创建 Python 文件

1. 打开你的编辑器（如 VS Code）

2. 新建文件，输入以下代码：

   ```python
   # hello.py
   print("Hello, World!")
   ```

3. 保存为 `hello.py`（文件名以 `.py` 结尾）

### 步骤 2：运行程序

- **方法一：命令行运行**

  ```bash
  python hello.py
  ```

  输出：

  ```
  Hello, World!
  ```

- **方法二：在 IDE 中运行**

  - VS Code：右键编辑器 → “Run Python File”
  - PyCharm：右键 → “Run 'hello'”

> 注意：
>
> - `print()` 是 Python 的内置函数，用于输出内容
> - 字符串必须用引号包围（单引号 `'` 或双引号 `"` 均可）
> - 代码区分大小写，`Print` 会报错

## 1.5 使用命令行与交互式解释器（REPL）

Python 提供**交互式解释器**（Read-Eval-Print Loop, REPL），可逐行执行代码，非常适合学习和测试。

### 启动 REPL

在终端或命令提示符中输入：

```bash
python
```

或（Linux/macOS）：

```bash
python3
```

你会看到类似提示：

```
Python 3.10.12 (main, ...)
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

### 在 REPL 中尝试

```python
>>> 2 + 3
5
>>> name = "Alice"
>>> print(f"Hello, {name}!")
Hello, Alice!
>>> exit()  # 退出 REPL
```

### REPL 的用途

- 快速测试表达式或函数
- 查看对象属性（如 `dir(str)`）
- 学习语言特性（无需创建文件）

> 小技巧：在 REPL 中按 ↑ 键可调出历史命令。
