# 第二章：基本语法与数据类型

本章介绍 Python 的基础语法规则和核心数据类型。掌握这些内容是编写任何 Python 程序的前提。

---

## 2.1 注释与代码风格（PEP 8 简介）

### 注释

注释用于解释代码，**不会被 Python 执行**。

- **单行注释**：以 `#` 开头

  ```python
  # 这是一个单行注释
  x = 5  # 行尾注释
  ```

- **多行注释**：使用三重引号（严格来说是字符串字面量，但常作注释用）

  ```python
  """
  这是一个多行注释，
  通常用于函数或模块的文档说明。
  """
  ```

>  建议：注释应说明“为什么”，而不是“做什么”（代码本身应自解释）。

### PEP 8 简介

[PEP 8](https://pep8.org/) 是 Python 官方代码风格指南，核心原则包括：

- **缩进**：使用 4 个空格（不是 Tab）
- **行长**：每行不超过 79 个字符
- **命名**：
  - 变量/函数：`snake_case`（小写字母+下划线），如 `user_name`
  - 类名：`PascalCase`，如 `MyClass`
- **空格**：
  - 逗号后加空格：`a, b, c`
  - 运算符两侧加空格：`x = 1 + 2`
  - 函数参数无空格：`func(a, b)`

> 工具推荐：使用 `black` 或 `autopep8` 自动格式化代码。

## 2.2 变量与命名规则

### 变量定义

Python 中变量无需声明类型，直接赋值即可：

```python
age = 25
name = "Bob"
```

### 命名规则

- **必须以字母（a-z, A-Z）或下划线 `_` 开头**
- **后续字符可为字母、数字（0-9）或下划线**
- **区分大小写**：`Age` 与 `age` 是不同变量
- **不能使用关键字**：如 `if`, `for`, `class` 等（可通过 `import keyword; print(keyword.kwlist)` 查看）

合法示例：

```python
user_name = "Alice"
_count = 0
PI = 3.14159  # 常量惯例：全大写
```

非法示例：

```python
1user = "error"   # 不能以数字开头
my-var = 10       # 不能包含连字符
class = "test"    # class 是关键字
```

> 注意：Python 没有真正的常量，`PI = 3.14` 仍可被修改，但按约定不应修改。

## 2.3 基本数据类型

Python 是动态类型语言，但每个值都有明确类型。初学者需重点掌握以下五种基本数据类型。

### 整数（`int`）

- 表示整数值，无大小限制（受限于内存）
- 支持多种进制：

  ```python
  decimal = 42        # 十进制
  binary = 0b101010   # 二进制（0b 前缀）
  octal = 0o52        # 八进制（0o 前缀）
  hex = 0x2A          # 十六进制（0x 前缀）
  ```

### 浮点数（`float`）

- 表示带小数点的数值，基于 IEEE 754 双精度
- 科学计数法：

  ```python
  pi = 3.14159
  large = 1.23e4  # 等价于 12300.0
  ```

> 注意：浮点数存在精度问题（如 `0.1 + 0.2 != 0.3`），金融计算建议用 `decimal` 模块。

### 布尔值（`bool`）

- 只有两个值：`True` 和 `False`（首字母大写！）
- 常用于条件判断：

  ```python
  is_active = True
  has_permission = False
  ```

>  提示：在数值上下文中，`True == 1`，`False == 0`。

### 字符串（`str`）

- 用单引号 `'` 或双引号 `"` 包围
- 多行字符串用三重引号：

  ```python
  s1 = 'Hello'
  s2 = "World"
  s3 = """This is a
  multi-line string."""
  ```

- 字符串是**不可变**的：任何“修改”操作都会创建新字符串

### 复数（`complex`）

- 表示数学中的复数，形式为 `a + bj`，其中 `a` 是实部，`b` 是虚部
- 虚数单位用 `j`（不是 `i`），且**必须写成 `bj` 形式**（如 `3j`，不能只写 `j`）

  ```python
  c1 = 3 + 4j
  c2 = 2.5j
  c3 = complex(1, 2)  # 等价于 1 + 2j
  ```

- 获取实部和虚部：

  ```python
  c = 3 + 4j
  print(c.real)  # 输出: 3.0
  print(c.imag)  # 输出: 4.0
  ```

> 应用场景：复数主要用于科学计算（如信号处理、物理仿真）。**日常编程中极少使用**，了解即可。

## 2.4 类型检查与转换

### 类型检查

使用 `type()` 查看变量类型：

```python
>>> type(42)
<class 'int'>
>>> type("hello")
<class 'str'>
>>> type(True)
<class 'bool'>
>>> type(1+2j)
<class 'complex'>
```

### 类型转换

通过内置函数进行显式转换：

| 转换目标 | 函数       | 示例                                  |
| -------- | ---------- | ------------------------------------- |
| 整数     | `int(x)`   | `int("123") → 123`                    |
| 浮点数   | `float(x)` | `float("3.14") → 3.14`                |
| 字符串   | `str(x)`   | `str(42) → "42"`                      |
| 布尔值   | `bool(x)`  | `bool(0) → False`, `bool("a") → True` |

> 注意：
> - **复数不能直接转为 `int` 或 `float`**（会报错）
> - 字符串转复数可用 `complex("3+4j")`，但格式必须严格

### 常见转换规则

- `bool(x)` 为 `False` 当且仅当 `x` 是：
  - `None`
  - `False`
  - 数值 `0`、`0.0`、`0j`
  - 空容器：`""`, `[]`, `{}`, `set()`
- 其他情况均为 `True`

## 2.5 动态类型与变量赋值机制

### 动态类型

Python 在运行时确定变量类型，同一变量可指向不同类型的对象：

```python
x = 10      # x 是 int
x = "ten"   # x 现在是 str
x = 1 + 2j  # x 现在是 complex
```

### 赋值机制

- **变量是对象的引用**，而非存储值本身
- 多变量赋值：

  ```python
  a = b = 10    # a 和 b 指向同一个整数对象
  x, y = 1, 2   # 元组解包
  ```

- 可变对象 vs 不可变对象：
  - **不可变**：`int`, `float`, `str`, `tuple`, `complex` — 修改会创建新对象
  - **可变**：`list`, `dict`, `set` — 修改原地生效

> 关键理解：Python 中一切皆对象，变量只是对象的标签。
