## 1.1 数据分析简介与应用场景

### 什么是数据分析？

数据分析是指用适当的统计分析方法对收集来的大量数据进行分析，提取有用信息和形成结论的过程。简单来说，就是从数据中发现规律、趋势和洞察。

### 数据分析的典型流程

1. **数据收集**：从数据库、文件、API、爬虫等来源获取数据
2. **数据清洗**：处理缺失值、异常值、重复值等问题
3. **数据探索**：了解数据分布、特征关系
4. **数据分析**：应用统计方法或模型进行分析
5. **数据可视化**：用图表展示分析结果
6. **报告输出**：形成结论和建议

### Python 在数据分析中的优势

- **丰富的库生态**：NumPy、Pandas、Matplotlib、Seaborn、Scikit-learn 等
- **简洁的语法**：代码量少，易于理解和维护
- **交互式编程**：Jupyter Notebook 支持即时反馈
- **社区活跃**：遇到问题容易找到解决方案
- **跨领域应用**：金融、电商、医疗、社交网络等

### 典型应用场景

- **商业智能（BI）**：销售分析、用户画像、市场趋势
- **金融分析**：股票预测、风险评估、量化交易
- **运营分析**：用户留存、转化漏斗、A/B 测试
- **科学研究**：实验数据处理、统计分析
- **网络运维**：日志分析、性能监控、异常检测

## 1.2 安装 Anaconda（推荐）或独立安装数据科学库

### 方案一：安装 Anaconda（强烈推荐初学者）

## 1.3 Jupyter Notebook/Lab 入门

### 什么是 Jupyter？

Jupyter 是一个交互式笔记本环境，允许你将代码、文本、图表整合在一个文档中，非常适合数据探索和教学。

- **Jupyter Notebook**：经典版本，界面简洁
- **JupyterLab**：新一代版本，功能更强大（推荐）

### 启动 Jupyter

#### 使用 Anaconda

```bash
# 启动 Jupyter Notebook
jupyter notebook

# 或启动 JupyterLab（推荐）
jupyter lab
```

浏览器会自动打开，默认地址为 `http://localhost:8888`

#### 使用 pip 安装

```bash
pip install jupyterlab
jupyter lab
```

### Jupyter 基本操作

#### 创建新笔记本

1. 点击 "New" → "Python 3"
2. 创建一个空白的 `.ipynb` 文件

#### 单元格类型

- **Code**：编写 Python 代码
- **Markdown**：编写说明文档（支持 Markdown 语法）
- **Raw**：原始文本（不渲染）

#### 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Shift + Enter` | 运行当前单元格，选中下一个 |
| `Ctrl + Enter` | 运行当前单元格，停留在当前 |
| `Alt + Enter` | 运行当前单元格，插入新单元格 |
| `A` | 在上方插入单元格 |
| `B` | 在下方插入单元格 |
| `D, D` | 删除当前单元格（按两次 D） |
| `M` | 转换为 Markdown 单元格 |
| `Y` | 转换为 Code 单元格 |

#### 第一个数据分析程序

在 Jupyter 中创建新单元格，输入：

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 创建示例数据
data = {
    '月份': ['1月', '2月', '3月', '4月', '5月'],
    '销售额': [12000, 15000, 13500, 18000, 22000]
}

df = pd.DataFrame(data)
print(df)

# 简单可视化
plt.figure(figsize=(10, 6))
plt.plot(df['月份'], df['销售额'], marker='o', linewidth=2)
plt.title('月度销售趋势')
plt.xlabel('月份')
plt.ylabel('销售额（元）')
plt.grid(True, alpha=0.3)
plt.show()
```

按 `Shift + Enter` 运行，即可看到表格和图表。

## 1.5 第一个数据分析程序

让我们完成一个完整的数据分析小项目，体验从数据加载到可视化的全流程。

### 项目：分析某商店月度销售数据

#### 步骤 1：准备数据

创建 CSV 文件 `sales_data.csv`：

```csv
月份,产品类别,销售额,订单数
1月,电子产品,50000,120
1月,服装,30000,200
1月,食品,20000,350
2月,电子产品,55000,130
2月,服装,32000,210
2月,食品,21000,360
3月,电子产品,60000,140
3月,服装,35000,220
3月,食品,23000,380
```

#### 步骤 2：编写分析代码

在 Jupyter 或 VS Code 中创建新 Notebook：

```python
# 导入必要的库
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# 设置中文显示（Windows）
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 1. 读取数据
df = pd.read_csv('sales_data.csv')
print("=== 数据预览 ===")
print(df.head())

# 2. 数据概览
print("\n=== 数据信息 ===")
print(df.info())

print("\n=== 统计摘要 ===")
print(df.describe())

# 3. 按月汇总销售额
monthly_sales = df.groupby('月份')['销售额'].sum().reset_index()
print("\n=== 月度销售汇总 ===")
print(monthly_sales)

# 4. 按产品类别分析
category_sales = df.groupby('产品类别')['销售额'].sum().reset_index()
print("\n=== 产品类别销售汇总 ===")
print(category_sales)

# 5. 可视化：月度销售趋势
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
plt.plot(monthly_sales['月份'], monthly_sales['销售额'], 
         marker='o', linewidth=2, color='steelblue')
plt.title('月度销售趋势', fontsize=14)
plt.xlabel('月份')
plt.ylabel('销售额（元）')
plt.grid(True, alpha=0.3)

# 6. 可视化：产品类别对比
plt.subplot(1, 2, 2)
plt.bar(category_sales['产品类别'], category_sales['销售额'], 
        color=['coral', 'skyblue', 'lightgreen'])
plt.title('产品类别销售对比', fontsize=14)
plt.xlabel('产品类别')
plt.ylabel('销售额（元）')
plt.grid(True, alpha=0.3, axis='y')

plt.tight_layout()
plt.savefig('sales_analysis.png', dpi=300, bbox_inches='tight')
plt.show()

print("\n分析完成！图表已保存为 sales_analysis.png")
```

#### 步骤 3：运行并查看结果

运行上述代码，你将看到：

- 数据的统计摘要
- 月度销售趋势图
- 产品类别对比图
- 保存的 PNG 图片文件

### 关键知识点总结

- **数据读取**：`pd.read_csv()` 读取 CSV 文件
- **数据探索**：`head()`, `info()`, `describe()` 快速了解数据
- **数据分组**：`groupby()` 进行聚合分析
- **数据可视化**：Matplotlib 绘制折线图和柱状图
- **图表保存**：`savefig()` 导出图片

## 常见问题

### Q1: Anaconda 占用空间太大怎么办？

可以使用 Miniconda（轻量版），只包含 conda 和 Python，需要时再安装库：

```bash
# 下载 Miniconda
# 访问 https://docs.conda.io/en/latest/miniconda.html

# 安装后手动安装所需库
conda install numpy pandas matplotlib jupyter
```

### Q2: Jupyter 无法显示中文怎么办？

在代码开头添加：

```python
import matplotlib.pyplot as plt

# Windows
plt.rcParams['font.sans-serif'] = ['SimHei']

# macOS
# plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']

# Linux
# plt.rcParams['font.sans-serif'] = ['WenQuanYi Micro Hei']

plt.rcParams['axes.unicode_minus'] = False
```

### Q3: 如何导出 Jupyter Notebook 为其他格式？

```bash
# 导出为 HTML
jupyter nbconvert --to html notebook.ipynb

# 导出为 PDF（需安装 LaTeX）
jupyter nbconvert --to pdf notebook.ipynb

# 导出为 Python 脚本
jupyter nbconvert --to python notebook.ipynb
```

或在 Jupyter 界面中：File → Download as → 选择格式
