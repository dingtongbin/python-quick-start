# 第五章：图论基础

本章将学习图（Graph）这一重要的非线性数据结构。图用于表示对象之间的关系，在社交网络、路由算法、推荐系统等领域有广泛应用。

---

## 5.1 图的基本概念

### 什么是图？

**图（Graph）** 是由顶点（Vertex）和边（Edge）组成的数据结构，用于表示对象之间的关系。

```
图 G = (V, E)
V = 顶点集合
E = 边集合
```

### 图的术语

```
    A --- B
    |   / |
    |  /  |
    | /   |
    C --- D

- 顶点（Vertex/Node）：A, B, C, D
- 边（Edge）：(A,B), (A,C), (B,C), (B,D), (C,D)
- 相邻（Adjacent）：A 和 B 相邻
- 度（Degree）：顶点的边数（A 的度是 2）
- 路径（Path）：从 A 到 D 的路径：A → B → D
- 环（Cycle）：A → B → C → A
```

### 图的类型

1. **无向图**：边没有方向
2. **有向图**：边有方向（箭头）
3. **加权图**：边有权重
4. **无权图**：边没有权重
5. **稠密图**：边的数量接近顶点数的平方
6. **稀疏图**：边的数量远小于顶点数的平方

## 5.2 图的表示方法

### 方法一：邻接矩阵

使用二维数组表示图，`matrix[i][j] = 1` 表示顶点 i 和 j 之间有边。

```python
class GraphAdjacencyMatrix:
    """邻接矩阵表示的图"""
    
    def __init__(self, vertices):
        self.vertices = vertices
        self.n = len(vertices)
        self.vertex_index = {v: i for i, v in enumerate(vertices)}
        self.matrix = [[0] * self.n for _ in range(self.n)]
    
    def add_edge(self, v1, v2, directed=False):
        """添加边"""
        i = self.vertex_index[v1]
        j = self.vertex_index[v2]
        
        self.matrix[i][j] = 1
        if not directed:
            self.matrix[j][i] = 1
    
    def remove_edge(self, v1, v2, directed=False):
        """删除边"""
        i = self.vertex_index[v1]
        j = self.vertex_index[v2]
        
        self.matrix[i][j] = 0
        if not directed:
            self.matrix[j][i] = 0
    
    def has_edge(self, v1, v2):
        """检查是否有边"""
        i = self.vertex_index[v1]
        j = self.vertex_index[v2]
        return self.matrix[i][j] == 1
    
    def get_neighbors(self, v):
        """获取邻居"""
        i = self.vertex_index[v]
        neighbors = []
        for j in range(self.n):
            if self.matrix[i][j] == 1:
                neighbors.append(self.vertices[j])
        return neighbors
    
    def display(self):
        """显示邻接矩阵"""
        print("   " + " ".join(self.vertices))
        for i, v in enumerate(self.vertices):
            print(f"{v}:  " + " ".join(map(str, self.matrix[i])))

# 测试
#   A --- B
#   |     |
#   C --- D
g = GraphAdjacencyMatrix(['A', 'B', 'C', 'D'])
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'D')

g.display()
#    A B C D
# A: 0 1 1 0
# B: 1 0 0 1
# C: 1 0 0 1
# D: 0 1 1 0

print("A 的邻居:", g.get_neighbors('A'))  # ['B', 'C']
print("A-B 有边吗?", g.has_edge('A', 'B'))  # True
```

**邻接矩阵的优缺点：**
- ✅ 判断两点间是否有边：O(1)
- ❌ 空间复杂度：O(V²)
- ❌ 遍历邻居：O(V)

### 方法二：邻接表

使用字典或列表存储每个顶点的邻居列表。

```python
from collections import defaultdict

class GraphAdjacencyList:
    """邻接表表示的图"""
    
    def __init__(self, directed=False):
        self.graph = defaultdict(list)
        self.directed = directed
    
    def add_edge(self, v1, v2, weight=None):
        """添加边"""
        if weight is not None:
            self.graph[v1].append((v2, weight))
        else:
            self.graph[v1].append(v2)
        
        if not self.directed:
            if weight is not None:
                self.graph[v2].append((v1, weight))
            else:
                self.graph[v2].append(v1)
    
    def remove_edge(self, v1, v2):
        """删除边"""
        if v2 in self.graph[v1]:
            self.graph[v1].remove(v2)
        
        if not self.directed and v1 in self.graph[v2]:
            self.graph[v2].remove(v1)
    
    def has_edge(self, v1, v2):
        """检查是否有边"""
        return v2 in self.graph[v1]
    
    def get_neighbors(self, v):
        """获取邻居"""
        return self.graph[v]
    
    def get_vertices(self):
        """获取所有顶点"""
        return list(self.graph.keys())
    
    def display(self):
        """显示图"""
        for vertex in self.graph:
            print(f"{vertex}: {self.graph[vertex]}")

# 测试
g = GraphAdjacencyList()
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'D')

g.display()
# A: ['B', 'C']
# B: ['A', 'D']
# C: ['A', 'D']
# D: ['B', 'C']

print("A 的邻居:", g.get_neighbors('A'))  # ['B', 'C']
```

**邻接表的优缺点：**
- ✅ 空间复杂度：O(V + E)
- ✅ 遍历邻居：O(度)
- ❌ 判断两点间是否有边：O(度)

### 表示方法对比

| 特性 | 邻接矩阵 | 邻接表 |
|------|----------|--------|
| 空间复杂度 | O(V²) | O(V + E) |
| 判断边存在 | O(1) | O(度) |
| 遍历邻居 | O(V) | O(度) |
| 添加边 | O(1) | O(1) |
| 适用场景 | 稠密图 | 稀疏图 |

## 5.3 图的遍历

### 深度优先搜索（DFS）

**原理：** 尽可能深地搜索图的分支，使用栈或递归实现。

```python
class GraphDFS:
    def __init__(self):
        self.graph = defaultdict(list)
    
    def add_edge(self, v1, v2):
        self.graph[v1].append(v2)
        self.graph[v2].append(v1)
    
    def dfs_recursive(self, start, visited=None):
        """DFS - 递归实现"""
        if visited is None:
            visited = set()
        
        visited.add(start)
        print(start, end=' ')
        
        for neighbor in self.graph[start]:
            if neighbor not in visited:
                self.dfs_recursive(neighbor, visited)
        
        return visited
    
    def dfs_iterative(self, start):
        """DFS - 迭代实现"""
        visited = set()
        stack = [start]
        
        while stack:
            vertex = stack.pop()
            
            if vertex not in visited:
                visited.add(vertex)
                print(vertex, end=' ')
                
                # 逆序添加，保证顺序
                for neighbor in reversed(self.graph[vertex]):
                    if neighbor not in visited:
                        stack.append(neighbor)
        
        return visited

# 测试
#   A --- B
#   |   / |
#   |  /  |
#   C --- D
g = GraphDFS()
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'D')

print("DFS 递归:", end=' ')
g.dfs_recursive('A')  # A B C D 或 A C B D
print("\nDFS 迭代:", end=' ')
g.dfs_iterative('A')
```

### 广度优先搜索（BFS）

**原理：** 逐层搜索，先访问离起点近的节点，使用队列实现。

```python
from collections import deque

class GraphBFS:
    def __init__(self):
        self.graph = defaultdict(list)
    
    def add_edge(self, v1, v2):
        self.graph[v1].append(v2)
        self.graph[v2].append(v1)
    
    def bfs(self, start):
        """BFS"""
        visited = set()
        queue = deque([start])
        visited.add(start)
        
        while queue:
            vertex = queue.popleft()
            print(vertex, end=' ')
            
            for neighbor in self.graph[vertex]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append(neighbor)
        
        return visited

# 测试
g = GraphBFS()
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'D')

print("\nBFS:", end=' ')
g.bfs('A')  # A B C D
```

### DFS vs BFS 对比

| 特性 | DFS | BFS |
|------|-----|-----|
| 数据结构 | 栈（递归） | 队列 |
| 空间复杂度 | O(V) | O(V) |
| 找最短路径 | ❌ | ✅（无权图） |
| 应用场景 | 拓扑排序、检测环 | 最短路径、层级遍历 |

## 5.4 最短路径算法

### Dijkstra 算法

用于求解**加权图**中单源最短路径，要求权重非负。

```python
import heapq

def dijkstra(graph, start):
    """
    Dijkstra 算法
    graph: 邻接表 {vertex: [(neighbor, weight), ...]}
    start: 起始顶点
    """
    # 初始化距离
    distances = {vertex: float('infinity') for vertex in graph}
    distances[start] = 0
    
    # 优先队列
    pq = [(0, start)]
    
    while pq:
        current_distance, current_vertex = heapq.heappop(pq)
        
        # 如果当前距离大于已知最短距离，跳过
        if current_distance > distances[current_vertex]:
            continue
        
        # 遍历邻居
        for neighbor, weight in graph[current_vertex]:
            distance = current_distance + weight
            
            # 如果找到更短的路径
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))
    
    return distances

# 测试
#   A --4-- B
#   |      /|
#   2    3  1
#   |  /    |
#   C --5-- D
graph = {
    'A': [('B', 4), ('C', 2)],
    'B': [('A', 4), ('C', 3), ('D', 1)],
    'C': [('A', 2), ('B', 3), ('D', 5)],
    'D': [('B', 1), ('C', 5)]
}

distances = dijkstra(graph, 'A')
print("从 A 出发的最短距离:")
for vertex, distance in distances.items():
    print(f"  A -> {vertex}: {distance}")
# A -> A: 0
# A -> B: 4
# A -> C: 2
# A -> D: 5 (A->B->D)
```

### Bellman-Ford 算法

可以处理**负权重**的图，并能检测负权环。

```python
def bellman_ford(graph, vertices, start):
    """
    Bellman-Ford 算法
    graph: 边列表 [(v1, v2, weight), ...]
    vertices: 所有顶点
    start: 起始顶点
    """
    # 初始化距离
    distances = {v: float('infinity') for v in vertices}
    distances[start] = 0
    
    # 松弛 V-1 次
    for _ in range(len(vertices) - 1):
        for v1, v2, weight in graph:
            if distances[v1] != float('infinity') and \
               distances[v1] + weight < distances[v2]:
                distances[v2] = distances[v1] + weight
    
    # 检测负权环
    for v1, v2, weight in graph:
        if distances[v1] != float('infinity') and \
           distances[v1] + weight < distances[v2]:
            raise ValueError("图中存在负权环")
    
    return distances

# 测试
vertices = ['A', 'B', 'C', 'D']
edges = [
    ('A', 'B', 4),
    ('A', 'C', 2),
    ('B', 'C', 3),
    ('B', 'D', 1),
    ('C', 'D', 5)
]

distances = bellman_ford(edges, vertices, 'A')
print("\nBellman-Ford 结果:")
for vertex, distance in distances.items():
    print(f"  A -> {vertex}: {distance}")
```

## 5.5 最小生成树

### Prim 算法

从一个顶点开始，逐步扩展生成树。

```python
def prim_mst(graph, start):
    """
    Prim 算法求最小生成树
    graph: 邻接表 {vertex: [(neighbor, weight), ...]}
    """
    mst = []
    visited = {start}
    
    # 优先队列：(weight, from_vertex, to_vertex)
    edges = [(weight, start, neighbor) 
             for neighbor, weight in graph[start]]
    heapq.heapify(edges)
    
    total_weight = 0
    
    while edges:
        weight, from_v, to_v = heapq.heappop(edges)
        
        if to_v in visited:
            continue
        
        visited.add(to_v)
        mst.append((from_v, to_v, weight))
        total_weight += weight
        
        # 添加新顶点的边
        for neighbor, w in graph[to_v]:
            if neighbor not in visited:
                heapq.heappush(edges, (w, to_v, neighbor))
    
    return mst, total_weight

# 测试
graph = {
    'A': [('B', 4), ('C', 2)],
    'B': [('A', 4), ('C', 3), ('D', 1)],
    'C': [('A', 2), ('B', 3), ('D', 5)],
    'D': [('B', 1), ('C', 5)]
}

mst, total_weight = prim_mst(graph, 'A')
print("\n最小生成树:")
for edge in mst:
    print(f"  {edge[0]} - {edge[1]}: {edge[2]}")
print(f"总权重: {total_weight}")
```

### Kruskal 算法

按权重从小到大选择边，避免形成环。

```python
class UnionFind:
    """并查集"""
    
    def __init__(self, vertices):
        self.parent = {v: v for v in vertices}
        self.rank = {v: 0 for v in vertices}
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])  # 路径压缩
        return self.parent[x]
    
    def union(self, x, y):
        root_x = self.find(x)
        root_y = self.find(y)
        
        if root_x == root_y:
            return False
        
        # 按秩合并
        if self.rank[root_x] < self.rank[root_y]:
            self.parent[root_x] = root_y
        elif self.rank[root_x] > self.rank[root_y]:
            self.parent[root_y] = root_x
        else:
            self.parent[root_y] = root_x
            self.rank[root_x] += 1
        
        return True

def kruskal_mst(vertices, edges):
    """
    Kruskal 算法求最小生成树
    edges: [(v1, v2, weight), ...]
    """
    # 按权重排序
    edges.sort(key=lambda x: x[2])
    
    uf = UnionFind(vertices)
    mst = []
    total_weight = 0
    
    for v1, v2, weight in edges:
        if uf.union(v1, v2):
            mst.append((v1, v2, weight))
            total_weight += weight
    
    return mst, total_weight

# 测试
vertices = ['A', 'B', 'C', 'D']
edges = [
    ('A', 'B', 4),
    ('A', 'C', 2),
    ('B', 'C', 3),
    ('B', 'D', 1),
    ('C', 'D', 5)
]

mst, total_weight = kruskal_mst(vertices, edges)
print("\nKruskal 最小生成树:")
for edge in mst:
    print(f"  {edge[0]} - {edge[1]}: {edge[2]}")
print(f"总权重: {total_weight}")
```

## 综合示例：社交网络分析

```python
from collections import defaultdict, deque

class SocialNetwork:
    """社交网络分析"""
    
    def __init__(self):
        self.graph = defaultdict(set)
    
    def add_friendship(self, person1, person2):
        """添加好友关系"""
        self.graph[person1].add(person2)
        self.graph[person2].add(person1)
    
    def get_friends(self, person):
        """获取好友列表"""
        return self.graph[person]
    
    def are_friends(self, person1, person2):
        """检查是否是好友"""
        return person2 in self.graph[person1]
    
    def find_shortest_path(self, start, end):
        """BFS 找最短路径"""
        if start == end:
            return [start]
        
        visited = {start}
        queue = deque([(start, [start])])
        
        while queue:
            current, path = queue.popleft()
            
            for friend in self.graph[current]:
                if friend == end:
                    return path + [friend]
                
                if friend not in visited:
                    visited.add(friend)
                    queue.append((friend, path + [friend]))
        
        return []  # 没有路径
    
    def get_mutual_friends(self, person1, person2):
        """获取共同好友"""
        return self.graph[person1] & self.graph[person2]
    
    def recommend_friends(self, person):
        """推荐好友（好友的好友）"""
        recommendations = set()
        
        for friend in self.graph[person]:
            for friend_of_friend in self.graph[friend]:
                if friend_of_friend != person and friend_of_friend not in self.graph[person]:
                    recommendations.add(friend_of_friend)
        
        return recommendations
    
    def get_connected_components(self):
        """获取连通分量"""
        visited = set()
        components = []
        
        for vertex in self.graph:
            if vertex not in visited:
                component = []
                queue = deque([vertex])
                visited.add(vertex)
                
                while queue:
                    current = queue.popleft()
                    component.append(current)
                    
                    for neighbor in self.graph[current]:
                        if neighbor not in visited:
                            visited.add(neighbor)
                            queue.append(neighbor)
                
                components.append(component)
        
        return components

# 测试
network = SocialNetwork()
network.add_friendship("Alice", "Bob")
network.add_friendship("Alice", "Charlie")
network.add_friendship("Bob", "David")
network.add_friendship("Charlie", "David")
network.add_friendship("David", "Eve")

print("Alice 的好友:", network.get_friends("Alice"))
print("Alice 和 Bob 是好友吗?", network.are_friends("Alice", "Bob"))
print("Alice 到 Eve 的最短路径:", network.find_shortest_path("Alice", "Eve"))
print("Alice 和 Bob 的共同好友:", network.get_mutual_friends("Alice", "Bob"))
print("推荐给 Alice 的好友:", network.recommend_friends("Alice"))
```
