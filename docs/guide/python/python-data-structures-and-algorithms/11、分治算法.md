# 第十一章：分治算法

分治算法将问题分成若干个子问题，分别求解后合并结果。

---

## 11.1 分治框架

```
分治三步曲：
1. 分解（Divide）：将问题分成子问题
2. 解决（Conquer）：递归求解子问题
3. 合并（Combine）：合并子问题的解
```

### 大整数乘法

```python
def karatsuba(x, y):
    """Karatsuba 大整数乘法"""
    if x < 10 or y < 10:
        return x * y
    
    n = max(len(str(x)), len(str(y)))
    m = n // 2
    
    high1, low1 = divmod(x, 10**m)
    high2, low2 = divmod(y, 10**m)
    
    z0 = karatsuba(low1, low2)
    z1 = karatsuba(high1 + low1, high2 + low2)
    z2 = karatsuba(high1, high2)
    
    return z2 * 10**(2*m) + (z1 - z2 - z0) * 10**m + z0

# 测试
print("1234 * 5678 =", karatsuba(1234, 5678))
```
