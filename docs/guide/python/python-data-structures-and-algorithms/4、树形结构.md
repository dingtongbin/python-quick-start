## 4.1 树的基本概念

### 什么是树？

**树（Tree）** 是一种非线性的层次数据结构，由节点（Node）和边（Edge）组成。

**树的定义：**
- 有一个特殊的节点称为**根节点**（Root）
- 除了根节点，每个节点有且只有一个父节点
- 每个节点可以有零个或多个子节点
- 没有子节点的节点称为**叶子节点**（Leaf）

### 树的术语

```
        A          ← 根节点（层级 0）
       / \
      B   C        ← 第 1 层
     / \   \
    D   E   F      ← 第 2 层（D, E, F 是叶子节点）
    
相关术语：
- 父节点（Parent）：B 是 D 和 E 的父节点
- 子节点（Child）：D 和 E 是 B 的子节点
- 兄弟节点（Sibling）：D 和 E 是兄弟节点
- 祖先（Ancestor）：A 是 D 的祖先
- 后代（Descendant）：D 是 A 的后代
- 深度（Depth）：节点到根的距离（D 的深度是 2）
- 高度（Height）：节点到最远叶子的距离（A 的高度是 2）
- 度（Degree）：节点的子节点数（B 的度是 2）
```

### 树的类型

1. **二叉树**：每个节点最多有两个子节点
2. **二叉搜索树**：左子节点 < 父节点 < 右子节点
3. **平衡二叉树**：左右子树高度差不超过 1
4. **完全二叉树**：除最后一层外，其他层都是满的
5. **满二叉树**：所有叶子节点都在同一层
6. **堆**：满足堆性质的完全二叉树

## 4.2 二叉树（Binary Tree）

### 二叉树的定义

**二叉树**是每个节点最多有两个子节点的树结构，通常称为左子节点和右子节点。

### 二叉树的实现

```python
class TreeNode:
    """二叉树节点"""
    
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class BinaryTree:
    """二叉树"""
    
    def __init__(self):
        self.root = None
    
    def insert(self, val):
        """插入节点（层序插入）"""
        new_node = TreeNode(val)
        
        if not self.root:
            self.root = new_node
            return
        
        # 使用队列进行层序遍历
        from collections import deque
        queue = deque([self.root])
        
        while queue:
            node = queue.popleft()
            
            if not node.left:
                node.left = new_node
                return
            else:
                queue.append(node.left)
            
            if not node.right:
                node.right = new_node
                return
            else:
                queue.append(node.right)
    
    def search(self, val):
        """搜索节点"""
        return self._search_recursive(self.root, val)
    
    def _search_recursive(self, node, val):
        if not node:
            return False
        if node.val == val:
            return True
        return (self._search_recursive(node.left, val) or 
                self._search_recursive(node.right, val))
    
    def delete(self, val):
        """删除节点（简化版：只删除叶子节点）"""
        if not self.root:
            return False
        
        if self.root.val == val and not self.root.left and not self.root.right:
            self.root = None
            return True
        
        return self._delete_helper(self.root, val)
    
    def _delete_helper(self, node, val):
        if not node:
            return False
        
        # 检查子节点
        if node.left and node.left.val == val:
            if not node.left.left and not node.left.right:
                node.left = None
                return True
        if node.right and node.right.val == val:
            if not node.right.left and not node.right.right:
                node.right = None
                return True
        
        return (self._delete_helper(node.left, val) or 
                self._delete_helper(node.right, val))
    
    def get_height(self):
        """获取树的高度"""
        return self._get_height(self.root)
    
    def _get_height(self, node):
        if not node:
            return 0
        return 1 + max(self._get_height(node.left), 
                      self._get_height(node.right))
    
    def get_size(self):
        """获取节点数量"""
        return self._get_size(self.root)
    
    def _get_size(self, node):
        if not node:
            return 0
        return 1 + self._get_size(node.left) + self._get_size(node.right)

# 测试
bt = BinaryTree()
for val in [1, 2, 3, 4, 5, 6, 7]:
    bt.insert(val)

print(f"树的高度: {bt.get_height()}")
print(f"节点数量: {bt.get_size()}")
print(f"搜索 5: {bt.search(5)}")
print(f"搜索 10: {bt.search(10)}")
```

### 二叉树的遍历

#### 1. 前序遍历（Pre-order）：根 → 左 → 右

```python
def preorder_traversal(root):
    """前序遍历 - 递归"""
    if not root:
        return []
    return [root.val] + preorder_traversal(root.left) + preorder_traversal(root.right)

def preorder_iterative(root):
    """前序遍历 - 迭代"""
    if not root:
        return []
    
    result = []
    stack = [root]
    
    while stack:
        node = stack.pop()
        result.append(node.val)
        
        # 先右后左（因为栈是后进先出）
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    
    return result

# 测试
#       1
#      / \
#     2   3
#    / \
#   4   5
root = TreeNode(1)
root.left = TreeNode(2)
root.right = TreeNode(3)
root.left.left = TreeNode(4)
root.left.right = TreeNode(5)

print("前序遍历:", preorder_traversal(root))  # [1, 2, 4, 5, 3]
```

#### 2. 中序遍历（In-order）：左 → 根 → 右

```python
def inorder_traversal(root):
    """中序遍历 - 递归"""
    if not root:
        return []
    return inorder_traversal(root.left) + [root.val] + inorder_traversal(root.right)

def inorder_iterative(root):
    """中序遍历 - 迭代"""
    result = []
    stack = []
    current = root
    
    while current or stack:
        # 到达最左节点
        while current:
            stack.append(current)
            current = current.left
        
        # 处理节点
        current = stack.pop()
        result.append(current.val)
        
        # 转向右子树
        current = current.right
    
    return result

print("中序遍历:", inorder_traversal(root))  # [4, 2, 5, 1, 3]
```

#### 3. 后序遍历（Post-order）：左 → 右 → 根

```python
def postorder_traversal(root):
    """后序遍历 - 递归"""
    if not root:
        return []
    return postorder_traversal(root.left) + postorder_traversal(root.right) + [root.val]

def postorder_iterative(root):
    """后序遍历 - 迭代"""
    if not root:
        return []
    
    result = []
    stack = [root]
    
    while stack:
        node = stack.pop()
        result.append(node.val)
        
        if node.left:
            stack.append(node.left)
        if node.right:
            stack.append(node.right)
    
    return result[::-1]  # 反转

print("后序遍历:", postorder_traversal(root))  # [4, 5, 2, 3, 1]
```

#### 4. 层序遍历（Level-order）：从上到下，从左到右

```python
from collections import deque

def level_order_traversal(root):
    """层序遍历"""
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        level_size = len(queue)
        level = []
        
        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)
            
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        result.append(level)
    
    return result

print("层序遍历:", level_order_traversal(root))
# [[1], [2, 3], [4, 5]]
```

### 遍历方式对比

| 遍历方式 | 顺序 | 应用场景 |
|----------|------|----------|
| 前序遍历 | 根左右 | 复制树、序列化 |
| 中序遍历 | 左根右 | BST 排序输出 |
| 后序遍历 | 左右根 | 删除树、计算表达式 |
| 层序遍历 | 从上到下 | BFS、最短路径 |

## 4.3 二叉搜索树（BST）

### BST 的定义

**二叉搜索树**（Binary Search Tree）是一种特殊的二叉树，满足以下性质：
- 左子树的所有节点值 < 根节点值
- 右子树的所有节点值 > 根节点值
- 左右子树也都是二叉搜索树

### BST 的实现

```python
class BSTNode:
    """BST 节点"""
    
    def __init__(self, val=0):
        self.val = val
        self.left = None
        self.right = None

class BinarySearchTree:
    """二叉搜索树"""
    
    def __init__(self):
        self.root = None
    
    def insert(self, val):
        """插入节点"""
        if not self.root:
            self.root = BSTNode(val)
        else:
            self._insert_recursive(self.root, val)
    
    def _insert_recursive(self, node, val):
        if val < node.val:
            if node.left:
                self._insert_recursive(node.left, val)
            else:
                node.left = BSTNode(val)
        elif val > node.val:
            if node.right:
                self._insert_recursive(node.right, val)
            else:
                node.right = BSTNode(val)
        # 如果相等，不插入（不允许重复）
    
    def search(self, val):
        """搜索节点"""
        return self._search_recursive(self.root, val)
    
    def _search_recursive(self, node, val):
        if not node:
            return False
        if val == node.val:
            return True
        elif val < node.val:
            return self._search_recursive(node.left, val)
        else:
            return self._search_recursive(node.right, val)
    
    def delete(self, val):
        """删除节点"""
        self.root = self._delete_recursive(self.root, val)
    
    def _delete_recursive(self, node, val):
        if not node:
            return None
        
        if val < node.val:
            node.left = self._delete_recursive(node.left, val)
        elif val > node.val:
            node.right = self._delete_recursive(node.right, val)
        else:
            # 找到要删除的节点
            
            # 情况1：叶子节点
            if not node.left and not node.right:
                return None
            
            # 情况2：只有一个子节点
            if not node.left:
                return node.right
            if not node.right:
                return node.left
            
            # 情况3：有两个子节点
            # 找到右子树的最小节点（中序后继）
            min_node = self._find_min(node.right)
            node.val = min_node.val
            node.right = self._delete_recursive(node.right, min_node.val)
        
        return node
    
    def _find_min(self, node):
        """找到最小节点"""
        current = node
        while current.left:
            current = current.left
        return current
    
    def find_min(self):
        """获取最小值"""
        if not self.root:
            return None
        min_node = self._find_min(self.root)
        return min_node.val
    
    def find_max(self):
        """获取最大值"""
        if not self.root:
            return None
        current = self.root
        while current.right:
            current = current.right
        return current.val
    
    def inorder(self):
        """中序遍历（有序输出）"""
        result = []
        self._inorder_recursive(self.root, result)
        return result
    
    def _inorder_recursive(self, node, result):
        if node:
            self._inorder_recursive(node.left, result)
            result.append(node.val)
            self._inorder_recursive(node.right, result)
    
    def get_height(self):
        """获取高度"""
        return self._get_height(self.root)
    
    def _get_height(self, node):
        if not node:
            return 0
        return 1 + max(self._get_height(node.left), 
                      self._get_height(node.right))

# 测试
bst = BinarySearchTree()
values = [50, 30, 70, 20, 40, 60, 80]
for val in values:
    bst.insert(val)

print("中序遍历:", bst.inorder())  # [20, 30, 40, 50, 60, 70, 80]
print("搜索 40:", bst.search(40))  # True
print("搜索 25:", bst.search(25))  # False
print("最小值:", bst.find_min())   # 20
print("最大值:", bst.find_max())   # 80
print("树高度:", bst.get_height())  # 3

# 删除节点
bst.delete(30)
print("删除 30 后:", bst.inorder())  # [20, 40, 50, 60, 70, 80]
```

### BST 的时间复杂度分析

| 操作 | 平均情况 | 最坏情况 | 说明 |
|------|----------|----------|------|
| 查找 | O(log n) | O(n) | 退化为链表 |
| 插入 | O(log n) | O(n) | 退化为链表 |
| 删除 | O(log n) | O(n) | 退化为链表 |

**最坏情况：** 当插入的元素已经有序时，BST 会退化为链表。

```python
# 最坏情况示例
bst_worst = BinarySearchTree()
for i in range(1, 8):  # 已排序
    bst_worst.insert(i)

print("最坏情况高度:", bst_worst.get_height())  # 7 (应该是 log2(7) ≈ 3)
```

## 4.4 平衡二叉树简介

### 为什么需要平衡？

BST 在最坏情况下会退化为链表，导致操作时间复杂度从 O(log n) 降到 O(n)。平衡二叉树通过保持树的平衡来避免这个问题。

### AVL 树

**AVL 树**是最早的自平衡二叉搜索树，要求每个节点的左右子树高度差不超过 1。

```python
class AVLNode:
    """AVL 树节点"""
    
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None
        self.height = 1

class AVLTree:
    """AVL 树"""
    
    def get_height(self, node):
        if not node:
            return 0
        return node.height
    
    def get_balance(self, node):
        """获取平衡因子"""
        if not node:
            return 0
        return self.get_height(node.left) - self.get_height(node.right)
    
    def rotate_right(self, y):
        """右旋转"""
        x = y.left
        T2 = x.right
        
        # 执行旋转
        x.right = y
        y.left = T2
        
        # 更新高度
        y.height = 1 + max(self.get_height(y.left), self.get_height(y.right))
        x.height = 1 + max(self.get_height(x.left), self.get_height(x.right))
        
        return x
    
    def rotate_left(self, x):
        """左旋转"""
        y = x.right
        T2 = y.left
        
        # 执行旋转
        y.left = x
        x.right = T2
        
        # 更新高度
        x.height = 1 + max(self.get_height(x.left), self.get_height(x.right))
        y.height = 1 + max(self.get_height(y.left), self.get_height(y.right))
        
        return y
    
    def insert(self, root, val):
        """插入节点并平衡"""
        # 标准 BST 插入
        if not root:
            return AVLNode(val)
        
        if val < root.val:
            root.left = self.insert(root.left, val)
        elif val > root.val:
            root.right = self.insert(root.right, val)
        else:
            return root  # 不允许重复
        
        # 更新高度
        root.height = 1 + max(self.get_height(root.left), 
                             self.get_height(root.right))
        
        # 获取平衡因子
        balance = self.get_balance(root)
        
        # LL 情况
        if balance > 1 and val < root.left.val:
            return self.rotate_right(root)
        
        # RR 情况
        if balance < -1 and val > root.right.val:
            return self.rotate_left(root)
        
        # LR 情况
        if balance > 1 and val > root.left.val:
            root.left = self.rotate_left(root.left)
            return self.rotate_right(root)
        
        # RL 情况
        if balance < -1 and val < root.right.val:
            root.right = self.rotate_right(root.right)
            return self.rotate_left(root)
        
        return root

# AVL 树能保持平衡，高度始终是 O(log n)
```

### 红黑树

**红黑树**是另一种自平衡二叉搜索树，广泛应用于各种库中（如 C++ STL 的 map/set，Java 的 TreeMap）。

**红黑树的性质：**
1. 每个节点是红色或黑色
2. 根节点是黑色
3. 所有叶子节点（NIL）是黑色
4. 如果一个节点是红色，则它的两个子节点都是黑色
5. 从任一节点到其每个叶子的所有路径都包含相同数量的黑色节点

红黑树的实现比较复杂，这里不做详细展开。Python 的 `dict` 和 `set` 内部使用了类似哈希表的结构，而不是红黑树。

## 4.5 堆（Heap）

### 堆的定义

**堆**是一种特殊的完全二叉树，满足堆性质：
- **最大堆**：父节点的值 ≥ 子节点的值
- **最小堆**：父节点的值 ≤ 子节点的值

### 堆的数组表示

```
        1              数组: [1, 3, 2, 7, 5, 4, 6]
       / \           索引:  0  1  2  3  4  5  6
      3   2
     / \ / \
    7  5 4  6

父子关系：
- 父节点 i 的左子节点: 2*i + 1
- 父节点 i 的右子节点: 2*i + 2
- 子节点 i 的父节点: (i-1) // 2
```

### 使用 heapq 模块

Python 提供了 `heapq` 模块来实现最小堆。

```python
import heapq

# 创建堆
heap = []

# 插入元素 - O(log n)
heapq.heappush(heap, 5)
heapq.heappush(heap, 3)
heapq.heappush(heap, 8)
heapq.heappush(heap, 1)

print("堆:", heap)  # [1, 3, 8, 5]

# 获取最小值 - O(1)
print("最小值:", heap[0])  # 1

# 弹出最小值 - O(log n)
min_val = heapq.heappop(heap)
print("弹出:", min_val)  # 1
print("堆:", heap)  # [3, 5, 8]

# 批量创建堆
nums = [5, 3, 8, 1, 9, 2]
heapq.heapify(nums)  # O(n)
print("堆化后:", nums)  # [1, 3, 2, 5, 9, 8]
```

### 实现最大堆

```python
import heapq

class MaxHeap:
    """最大堆（通过取负数实现）"""
    
    def __init__(self):
        self.heap = []
    
    def push(self, val):
        heapq.heappush(self.heap, -val)
    
    def pop(self):
        return -heapq.heappop(self.heap)
    
    def peek(self):
        return -self.heap[0] if self.heap else None
    
    def __len__(self):
        return len(self.heap)

# 测试
max_heap = MaxHeap()
for val in [5, 3, 8, 1, 9]:
    max_heap.push(val)

print("最大值:", max_heap.peek())  # 9
print("弹出:", max_heap.pop())     # 9
print("弹出:", max_heap.pop())     # 8
```

### 堆的应用：Top K 问题

```python
def top_k_elements(nums, k):
    """找出最大的 K 个元素"""
    if k <= 0:
        return []
    
    # 使用最小堆维护最大的 K 个元素
    min_heap = nums[:k]
    heapq.heapify(min_heap)
    
    for num in nums[k:]:
        if num > min_heap[0]:
            heapq.heapreplace(min_heap, num)
    
    return sorted(min_heap, reverse=True)

# 测试
nums = [3, 1, 5, 12, 2, 11, 9, 7, 6, 8]
k = 3
print(f"最大的 {k} 个元素:", top_k_elements(nums, k))  # [12, 11, 9]
```

### 堆排序

```python
def heap_sort(nums):
    """堆排序"""
    # 创建最大堆
    max_heap = []
    for num in nums:
        heapq.heappush(max_heap, -num)
    
    # 依次弹出
    result = []
    while max_heap:
        result.append(-heapq.heappop(max_heap))
    
    return result

# 测试
nums = [5, 3, 8, 1, 9, 2]
print("排序前:", nums)
print("排序后:", heap_sort(nums))  # [1, 2, 3, 5, 8, 9]
```

## 综合示例：实现完整的二叉搜索树

```python
class BST:
    """完整的二叉搜索树实现"""
    
    def __init__(self):
        self.root = None
        self.size = 0
    
    class Node:
        def __init__(self, val):
            self.val = val
            self.left = None
            self.right = None
    
    def insert(self, val):
        """插入"""
        self.root = self._insert(self.root, val)
        self.size += 1
    
    def _insert(self, node, val):
        if not node:
            return self.Node(val)
        
        if val < node.val:
            node.left = self._insert(node.left, val)
        elif val > node.val:
            node.right = self._insert(node.right, val)
        
        return node
    
    def search(self, val):
        """搜索"""
        return self._search(self.root, val)
    
    def _search(self, node, val):
        if not node:
            return False
        if val == node.val:
            return True
        elif val < node.val:
            return self._search(node.left, val)
        else:
            return self._search(node.right, val)
    
    def delete(self, val):
        """删除"""
        self.root = self._delete(self.root, val)
        self.size -= 1
    
    def _delete(self, node, val):
        if not node:
            return None
        
        if val < node.val:
            node.left = self._delete(node.left, val)
        elif val > node.val:
            node.right = self._delete(node.right, val)
        else:
            if not node.left:
                return node.right
            if not node.right:
                return node.left
            
            # 找到中序后继
            successor = self._find_min(node.right)
            node.val = successor.val
            node.right = self._delete(node.right, successor.val)
        
        return node
    
    def _find_min(self, node):
        while node.left:
            node = node.left
        return node
    
    def inorder(self):
        """中序遍历"""
        result = []
        self._inorder(self.root, result)
        return result
    
    def _inorder(self, node, result):
        if node:
            self._inorder(node.left, result)
            result.append(node.val)
            self._inorder(node.right, result)
    
    def get_min(self):
        """最小值"""
        if not self.root:
            return None
        return self._find_min(self.root).val
    
    def get_max(self):
        """最大值"""
        if not self.root:
            return None
        node = self.root
        while node.right:
            node = node.right
        return node.val
    
    def get_height(self):
        """高度"""
        return self._get_height(self.root)
    
    def _get_height(self, node):
        if not node:
            return 0
        return 1 + max(self._get_height(node.left), 
                      self._get_height(node.right))
    
    def __len__(self):
        return self.size
    
    def __contains__(self, val):
        return self.search(val)
    
    def __repr__(self):
        return f"BST({self.inorder()})"
```

### 练习 1：验证二叉搜索树

给定一个二叉树，判断其是否是有效的二叉搜索树。

```python
def is_valid_bst(root):
    def validate(node, lower=float('-inf'), upper=float('inf')):
        if not node:
            return True
        
        val = node.val
        if val <= lower or val >= upper:
            return False
        
        return (validate(node.left, lower, val) and 
                validate(node.right, val, upper))
    
    return validate(root)
```

```python
def max_depth(root):
    if not root:
        return 0
    return 1 + max(max_depth(root.left), max_depth(root.right))
```

### 练习 3：合并两个二叉树

```python
def merge_trees(t1, t2):
    if not t1:
        return t2
    if not t2:
        return t1
    
    t1.val += t2.val
    t1.left = merge_trees(t1.left, t2.left)
    t1.right = merge_trees(t1.right, t2.right)
    
    return t1
```
