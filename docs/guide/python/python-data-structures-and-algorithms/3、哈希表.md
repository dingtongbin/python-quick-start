# 第三章：哈希表

本章将深入学习哈希表（Hash Table）这一重要的数据结构。哈希表通过哈希函数实现快速的查找、插入和删除操作，是实际应用中最常用的数据结构之一。

---

## 3.1 哈希表基本概念

### 什么是哈希表？

**哈希表**（Hash Table），也叫散列表，是一种通过哈希函数将键（Key）映射到数组索引的数据结构，从而实现平均 O(1) 时间复杂度的查找、插入和删除操作。

**核心思想：**
```
键（Key） → 哈希函数 → 数组索引 → 存储值（Value）
```

### 哈希表的特点

✅ **优点：**
- 查找速度快：平均 O(1)
- 插入速度快：平均 O(1)
- 删除速度快：平均 O(1)
- 实现相对简单

❌ **缺点：**
- 需要额外的空间
- 不支持有序遍历
- 哈希冲突需要处理
- 最坏情况退化为 O(n)

### 哈希表的组成

```python
"""
哈希表 = 数组 + 哈希函数 + 冲突解决方法

数组：存储数据的容器
哈希函数：将键转换为数组索引
冲突解决：处理多个键映射到同一索引的情况
"""
```

## 3.2 哈希函数设计

### 哈希函数的要求

一个好的哈希函数应该满足：

1. **确定性**：相同的键总是产生相同的哈希值
2. **均匀性**：不同的键尽可能均匀分布
3. **高效性**：计算速度快
4. **低冲突率**：尽量减少不同键映射到同一索引

### 常见哈希函数

#### 1. 除留余数法

```python
def hash_function(key, table_size):
    """除留余数法"""
    return key % table_size

# 示例
print(hash_function(10, 7))  # 3
print(hash_function(17, 7))  # 3 (冲突！)
print(hash_function(24, 7))  # 3 (冲突！)
```

#### 2. 乘法哈希

```python
def multiplicative_hash(key, table_size):
    """乘法哈希函数"""
    A = 0.6180339887  # 黄金比例的倒数
    fraction = (key * A) % 1
    return int(fraction * table_size)

# 示例
for i in range(10):
    print(f"{i} -> {multiplicative_hash(i, 10)}")
```

#### 3. 字符串哈希

```python
def string_hash(key, table_size):
    """字符串哈希函数"""
    hash_value = 0
    for char in key:
        hash_value = (hash_value * 31 + ord(char)) % table_size
    return hash_value

# 示例
print(string_hash("apple", 10))
print(string_hash("banana", 10))
print(string_hash("orange", 10))
```

#### 4. Python 内置哈希

```python
# Python 内置的 hash() 函数
print(hash("hello"))      # 字符串
print(hash(123))          # 整数
print(hash(3.14))         # 浮点数
print(hash((1, 2, 3)))    # 元组

# 注意：可变对象不能哈希
# hash([1, 2, 3])  # TypeError!
```

## 3.3 冲突解决方法

### 什么是哈希冲突？

当两个或多个不同的键通过哈希函数映射到同一个数组索引时，就发生了**哈希冲突**。

```python
# 示例：冲突
table_size = 7
key1 = 10  # 10 % 7 = 3
key2 = 17  # 17 % 7 = 3  ← 冲突！
key3 = 24  # 24 % 7 = 3  ← 冲突！
```

### 方法一：链地址法（Separate Chaining）

**原理：** 每个数组位置存储一个链表，所有映射到该位置的键值对都存储在链表中。

```python
class HashTableChaining:
    """使用链地址法解决冲突的哈希表"""
    
    def __init__(self, size=10):
        self.size = size
        self.table = [[] for _ in range(size)]  # 每个位置是一个列表
        self.count = 0
    
    def _hash(self, key):
        """哈希函数"""
        if isinstance(key, str):
            # 字符串哈希
            hash_value = 0
            for char in key:
                hash_value = (hash_value * 31 + ord(char)) % self.size
            return hash_value
        else:
            return key % self.size
    
    def put(self, key, value):
        """插入或更新键值对"""
        index = self._hash(key)
        
        # 检查是否已存在
        for i, (k, v) in enumerate(self.table[index]):
            if k == key:
                self.table[index][i] = (key, value)  # 更新
                return
        
        # 新增
        self.table[index].append((key, value))
        self.count += 1
        
        # 负载因子超过 0.75，扩容
        if self.count / self.size > 0.75:
            self._resize()
    
    def get(self, key):
        """获取值"""
        index = self._hash(key)
        
        for k, v in self.table[index]:
            if k == key:
                return v
        
        raise KeyError(f"Key '{key}' not found")
    
    def delete(self, key):
        """删除键值对"""
        index = self._hash(key)
        
        for i, (k, v) in enumerate(self.table[index]):
            if k == key:
                del self.table[index][i]
                self.count -= 1
                return v
        
        raise KeyError(f"Key '{key}' not found")
    
    def contains(self, key):
        """检查键是否存在"""
        index = self._hash(key)
        
        for k, v in self.table[index]:
            if k == key:
                return True
        
        return False
    
    def _resize(self):
        """扩容"""
        old_table = self.table
        self.size *= 2
        self.table = [[] for _ in range(self.size)]
        self.count = 0
        
        # 重新哈希所有元素
        for bucket in old_table:
            for key, value in bucket:
                self.put(key, value)
    
    def __getitem__(self, key):
        return self.get(key)
    
    def __setitem__(self, key, value):
        self.put(key, value)
    
    def __delitem__(self, key):
        self.delete(key)
    
    def __contains__(self, key):
        return self.contains(key)
    
    def __len__(self):
        return self.count
    
    def __repr__(self):
        items = []
        for i, bucket in enumerate(self.table):
            if bucket:
                items.append(f"  [{i}]: {bucket}")
        return f"HashTable(\n{', '.join(items)}\n)"

# 测试
ht = HashTableChaining(5)
ht["apple"] = 5
ht["banana"] = 3
ht["orange"] = 8
ht["grape"] = 10

print(ht)
print(f"apple: {ht['apple']}")
print(f"banana: {ht['banana']}")
print(f"'pear' in ht: {'pear' in ht}")
```

### 方法二：开放寻址法（Open Addressing）

**原理：** 所有元素都存储在数组中，发生冲突时寻找下一个空位。

#### 线性探测

```python
class HashTableLinearProbing:
    """使用线性探测的哈希表"""
    
    def __init__(self, size=10):
        self.size = size
        self.keys = [None] * size
        self.values = [None] * size
        self.count = 0
    
    def _hash(self, key):
        if isinstance(key, str):
            hash_value = 0
            for char in key:
                hash_value = (hash_value * 31 + ord(char)) % self.size
            return hash_value
        else:
            return key % self.size
    
    def put(self, key, value):
        """插入或更新"""
        if self.count / self.size > 0.7:
            self._resize()
        
        index = self._hash(key)
        
        # 线性探测
        while self.keys[index] is not None:
            if self.keys[index] == key:
                self.values[index] = value  # 更新
                return
            index = (index + 1) % self.size
        
        # 找到空位
        self.keys[index] = key
        self.values[index] = value
        self.count += 1
    
    def get(self, key):
        """获取值"""
        index = self._hash(key)
        
        while self.keys[index] is not None:
            if self.keys[index] == key:
                return self.values[index]
            index = (index + 1) % self.size
        
        raise KeyError(f"Key '{key}' not found")
    
    def delete(self, key):
        """删除（使用墓碑标记）"""
        index = self._hash(key)
        
        while self.keys[index] is not None:
            if self.keys[index] == key:
                self.keys[index] = "<DELETED>"  # 墓碑标记
                self.values[index] = None
                self.count -= 1
                return
            index = (index + 1) % self.size
        
        raise KeyError(f"Key '{key}' not found")
    
    def _resize(self):
        """扩容"""
        old_keys = self.keys
        old_values = self.values
        
        self.size *= 2
        self.keys = [None] * self.size
        self.values = [None] * self.size
        self.count = 0
        
        for key, value in zip(old_keys, old_values):
            if key is not None and key != "<DELETED>":
                self.put(key, value)
    
    def __getitem__(self, key):
        return self.get(key)
    
    def __setitem__(self, key, value):
        self.put(key, value)
    
    def __repr__(self):
        items = []
        for i in range(self.size):
            if self.keys[i] is not None:
                items.append(f"  [{i}]: {self.keys[i]} -> {self.values[i]}")
        return f"HashTable(\n{', '.join(items)}\n)"

# 测试
ht = HashTableLinearProbing(5)
ht[10] = "A"
ht[15] = "B"  # 15 % 5 = 0, 冲突，放到位置 1
ht[20] = "C"  # 20 % 5 = 0, 冲突，放到位置 2

print(ht)
```

#### 二次探测

```python
def quadratic_probing(hash_table, key, attempt):
    """二次探测：index = (hash + i²) % size"""
    base_index = hash(key)
    return (base_index + attempt * attempt) % len(hash_table)
```

#### 双重哈希

```python
def double_hashing(hash_table, key, attempt):
    """双重哈希：index = (hash1 + i * hash2) % size"""
    hash1 = key % len(hash_table)
    hash2 = 7 - (key % 7)  # 第二个哈希函数
    return (hash1 + attempt * hash2) % len(hash_table)
```

### 冲突解决方法对比

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 链地址法 | 实现简单，不会聚集 | 需要额外空间存储指针 | 通用场景 |
| 线性探测 | 缓存友好 | 一次聚集问题 | 负载因子低时 |
| 二次探测 | 减少聚集 | 计算稍复杂 | 中等负载 |
| 双重哈希 | 分布均匀 | 需要两个哈希函数 | 高负载场景 |

```python
def word_frequency(text):
    """统计词频"""
    words = text.lower().split()
    freq = {}
    
    for word in words:
```

### 应用 2：两数之和

```python
def two_sum(nums, target):
    """
    LeetCode 1. 两数之和
    时间复杂度: O(n)
    空间复杂度: O(n)
    """
    hash_map = {}
    
    for i, num in enumerate(nums):
        complement = target - num
        
        if complement in hash_map:
            return [hash_map[complement], i]
        
        hash_map[num] = i
    
    return []

# 测试
```

### 应用 3：缓存系统

```python
class LRUCache:
    """LRU 缓存（最近最少使用）"""
    
    def __init__(self, capacity):
        self.capacity = capacity
        self.cache = {}
        self.order = []  # 记录访问顺序
    
    def get(self, key):
        if key in self.cache:
            # 移动到最新位置
            self.order.remove(key)
            self.order.append(key)
            return self.cache[key]
        return -1
    
    def put(self, key, value):
        if key in self.cache:
            self.order.remove(key)
        elif len(self.cache) >= self.capacity:
            # 删除最久未使用的
            oldest = self.order.pop(0)
            del self.cache[oldest]
        
        self.cache[key] = value
        self.order.append(key)

# 测试
cache = LRUCache(2)
cache.put(1, 1)
cache.put(2, 2)
print(cache.get(1))       # 返回 1
cache.put(3, 3)           # 删除 key 2
print(cache.get(2))       # 返回 -1 (未找到)
```

### 应用 4：分组数据

```python
from collections import defaultdict

def group_anagrams(strs):
    """
    LeetCode 49. 字母异位词分组
    """
    groups = defaultdict(list)
    
    for s in strs:
        # 排序后的字符串作为键
        key = ''.join(sorted(s))
        groups[key].append(s)
    
    return list(groups.values())

# 测试

print(group_anagrams(strs))
```

## 综合示例：实现完整的哈希表

```python
class HashMap:
    """完整的哈希表实现"""
    
    def __init__(self, initial_capacity=16):
        self.capacity = initial_capacity
        self.size = 0
        self.load_factor_threshold = 0.75
        self.buckets = [[] for _ in range(self.capacity)]
    
    def _hash(self, key):
        """哈希函数"""
        if isinstance(key, str):
            # DJB2 哈希算法
            hash_value = 5381
            for char in key:
                hash_value = ((hash_value << 5) + hash_value) + ord(char)
                hash_value = hash_value & 0xFFFFFFFF  # 保持32位
            return hash_value % self.capacity
        else:
            return hash(key) % self.capacity
    
    def put(self, key, value):
        """插入或更新"""
        if self.size / self.capacity >= self.load_factor_threshold:
            self._resize()
        
        index = self._hash(key)
        bucket = self.buckets[index]
        
        # 检查是否已存在
        for i, (k, v) in enumerate(bucket):
            if k == key:
                bucket[i] = (key, value)
                return
        
        # 新增
        bucket.append((key, value))
        self.size += 1
    
    def get(self, key, default=None):
        """获取值"""
        index = self._hash(key)
        bucket = self.buckets[index]
        
        for k, v in bucket:
            if k == key:
                return v
        
        return default
    
    def delete(self, key):
        """删除"""
        index = self._hash(key)
        bucket = self.buckets[index]
        
        for i, (k, v) in enumerate(bucket):
            if k == key:
                del bucket[i]
                self.size -= 1
                return v
        
        raise KeyError(f"Key '{key}' not found")
    
    def contains(self, key):
        """检查是否存在"""
        return self.get(key) is not None
    
    def keys(self):
        """获取所有键"""
        result = []
        for bucket in self.buckets:
            for k, v in bucket:
                result.append(k)
        return result
    
    def values(self):
        """获取所有值"""
        result = []
        for bucket in self.buckets:
            for k, v in bucket:
                result.append(v)
        return result
    
    def items(self):
        """获取所有键值对"""
        result = []
        for bucket in self.buckets:
            for k, v in bucket:
                result.append((k, v))
        return result
    
    def _resize(self):
        """扩容"""
        old_buckets = self.buckets
        self.capacity *= 2
        self.size = 0
        self.buckets = [[] for _ in range(self.capacity)]
        
        for bucket in old_buckets:
            for key, value in bucket:
                self.put(key, value)
    
    def __getitem__(self, key):
        value = self.get(key)
        if value is None and not self.contains(key):
            raise KeyError(f"Key '{key}' not found")
        return value
    
    def __setitem__(self, key, value):
        self.put(key, value)
    
    def __delitem__(self, key):
        self.delete(key)
    
    def __contains__(self, key):
        return self.contains(key)
    
    def __len__(self):
        return self.size
    
    def __repr__(self):
        items = ', '.join(f'{k}: {v}' for k, v in self.items())
        return f'{{{items}}}'

# 测试
hm = HashMap()
hm["name"] = "Alice"
hm["age"] = 25
hm["city"] = "Beijing"

print(hm)
print(f"name: {hm['name']}")
print(f"age: {hm['age']}")
print(f"Keys: {hm.keys()}")
print(f"Size: {len(hm)}")

# 性能测试

for i in range(100000):
    hm[i] = i * 2
end = time.time()

print(f"\n插入 100000 个元素耗时: {(end-start)*1000:.2f}ms")
print(f"最终大小: {len(hm)}")
```

### 练习 2：最长无重复字符子串

给定一个字符串，找出其中不含有重复字符的最长子串的长度。

```python
def length_of_longest_substring(s):
    """
    LeetCode 3. 无重复字符的最长子串
    时间复杂度: O(n)
    """
    char_map = {}
    left = 0
    max_length = 0
    
    for right, char in enumerate(s):
        if char in char_map and char_map[char] >= left:
            left = char_map[char] + 1
        char_map[char] = right
        max_length = max(max_length, right - left + 1)
    
    return max_length
```
