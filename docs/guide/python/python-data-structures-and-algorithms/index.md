# 🧮 数据结构与算法：提升编程思维

算法是编程的灵魂。本教程用 Python 讲解常用数据结构和算法，帮你建立系统的算法思维，从容应对技术面试。

所有示例基于 **Python 3.10**，从概念到代码，强调理解而非死记硬背。

---

## 💡 学习建议

> **理解思想比背代码重要。** 搞懂每种算法的核心思路，代码自然写得出来。

- ⏱️ 预计学习时间：**4-8 周**（每天 1-2 小时）
- 🎯 学完能做什么：通过中等难度算法面试、解决复杂编程问题
- 📌 建议配合刷题：[LeetCode](https://leetcode.cn/) 对应章节练习

## ❓ 常见问题

**Q: 需要先学完 Python 基础吗？**
A: 是的，至少要熟悉基本语法、函数、列表和字典。建议先学完 [Python 基础](/guide/python/python-basics/) 前 7 章。

**Q: 看完就能通过面试吗？**
A: 看完能理解算法思想，但面试还需要刷题练习。建议每学完一章就在 LeetCode 上做 3-5 道对应题目。
