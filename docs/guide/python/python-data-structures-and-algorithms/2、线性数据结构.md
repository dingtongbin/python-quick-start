## 2.1 数组（Array）

### 数组的基本概念

**数组**是一种连续的内存空间存储相同类型数据的数据结构。

**特点：**
- ✅ **随机访问**：通过索引 O(1) 时间访问任意元素
- ✅ **缓存友好**：连续内存，CPU 缓存命中率高
- ❌ **固定大小**：传统数组大小固定（Python 列表是动态数组）
- ❌ **插入删除慢**：需要移动元素，O(n) 时间

### Python 列表的实现原理

Python 的 list 实际上是**动态数组**：
- 自动扩容：当空间不足时，分配更大的内存并复制数据
- 可以存储不同类型的元素
- 支持动态增删

```python
# 数组基本操作
arr = [1, 2, 3, 4, 5]

# 访问 - O(1)
print(arr[0])       # 1
print(arr[-1])      # 5

# 修改 - O(1)
arr[2] = 10
print(arr)          # [1, 2, 10, 4, 5]

# 尾部添加 - 平均 O(1)
arr.append(6)
print(arr)          # [1, 2, 10, 4, 5, 6]

# 指定位置插入 - O(n)
arr.insert(2, 99)
print(arr)          # [1, 2, 99, 10, 4, 5, 6]

# 删除 - O(n)
arr.remove(99)
print(arr)          # [1, 2, 10, 4, 5, 6]

# 切片 - O(k)，k为切片长度
print(arr[1:4])     # [2, 10, 4]
```

### 数组操作的时间复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 访问元素 | O(1) | 通过索引直接访问 |
| 搜索元素 | O(n) | 可能需要遍历整个数组 |
| 插入（末尾） | O(1)* | 平均情况，偶尔需要扩容 |
| 插入（中间） | O(n) | 需要移动后续元素 |
| 删除（末尾） | O(1) | 直接删除 |
| 删除（中间） | O(n) | 需要移动后续元素 |

### 数组应用实例

#### 示例 1：两数之和

```python
def two_sum(nums, target):
    """
    找出数组中两个数之和等于目标值的索引
    时间复杂度: O(n)
    空间复杂度: O(n)
    """
    hash_map = {}
    
    for i, num in enumerate(nums):
        complement = target - num
        if complement in hash_map:
            return [hash_map[complement], i]
        hash_map[num] = i
    
    return []

# 测试
nums = [2, 7, 11, 15]
target = 9
print(f"两数之和: {two_sum(nums, target)}")  # [0, 1]
```

#### 示例 2：最大子数组和（Kadane's Algorithm）

```python
def max_subarray_sum(nums):
    """
    找出连续子数组的最大和
    时间复杂度: O(n)
    空间复杂度: O(1)
    """
    if not nums:
        return 0
    
    max_sum = current_sum = nums[0]
    
    for num in nums[1:]:
        # 要么从当前元素重新开始，要么继续累加
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)
    
    return max_sum

# 测试
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(f"最大子数组和: {max_subarray_sum(nums)}")  # 6 (子数组 [4,-1,2,1])
```

#### 示例 3：旋转数组

```python
def rotate_array(nums, k):
    """
    将数组向右旋转 k 步
    时间复杂度: O(n)
    空间复杂度: O(1)
    """
    n = len(nums)
    k = k % n  # 处理 k > n 的情况
    
    # 三次反转法
    def reverse(left, right):
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
    
    reverse(0, n - 1)      # 反转整个数组
    reverse(0, k - 1)      # 反转前 k 个元素
    reverse(k, n - 1)      # 反转剩余元素

# 测试
nums = [1, 2, 3, 4, 5, 6, 7]
rotate_array(nums, 3)
print(f"旋转后: {nums}")  # [5, 6, 7, 1, 2, 3, 4]
```

## 2.2 链表（Linked List）

### 链表的概念

**链表**是一种非连续的线性数据结构，由一系列节点组成，每个节点包含数据和指向下一个节点的指针。

**链表 vs 数组：**

| 特性 | 数组 | 链表 |
|------|------|------|
| 内存 | 连续 | 不连续 |
| 访问 | O(1) | O(n) |
| 插入/删除 | O(n) | O(1)* |
| 大小 | 固定/动态 | 动态 |
| 缓存性能 | 好 | 差 |

*前提是已知节点位置

### 单链表的实现

```python
class ListNode:
    """链表节点"""
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class SinglyLinkedList:
    """单链表实现"""
    
    def __init__(self):
        self.head = None
        self.size = 0
    
    def get(self, index):
        """获取指定位置的元素 - O(n)"""
        if index < 0 or index >= self.size:
            return -1
        
        current = self.head
        for _ in range(index):
            current = current.next
        
        return current.val
    
    def add_at_head(self, val):
        """在头部添加元素 - O(1)"""
        new_node = ListNode(val)
        new_node.next = self.head
        self.head = new_node
        self.size += 1
    
    def add_at_tail(self, val):
        """在尾部添加元素 - O(n)"""
        new_node = ListNode(val)
        
        if not self.head:
            self.head = new_node
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = new_node
        
        self.size += 1
    
    def add_at_index(self, index, val):
        """在指定位置添加元素 - O(n)"""
        if index < 0 or index > self.size:
            return
        
        if index == 0:
            self.add_at_head(val)
            return
        
        new_node = ListNode(val)
        current = self.head
        
        for _ in range(index - 1):
            current = current.next
        
        new_node.next = current.next
        current.next = new_node
        self.size += 1
    
    def delete_at_index(self, index):
        """删除指定位置的元素 - O(n)"""
        if index < 0 or index >= self.size:
            return
        
        if index == 0:
            self.head = self.head.next
        else:
            current = self.head
            for _ in range(index - 1):
                current = current.next
            current.next = current.next.next
        
        self.size -= 1
    
    def display(self):
        """显示链表"""
        elements = []
        current = self.head
        while current:
            elements.append(str(current.val))
            current = current.next
        print(" -> ".join(elements) + " -> None")

# 测试
linked_list = SinglyLinkedList()
linked_list.add_at_tail(1)
linked_list.add_at_tail(2)
linked_list.add_at_tail(3)
linked_list.add_at_head(0)
print("链表:", end=" ")
linked_list.display()  # 0 -> 1 -> 2 -> 3 -> None

print(f"索引1的值: {linked_list.get(1)}")  # 1

linked_list.delete_at_index(2)
print("删除后:", end=" ")
linked_list.display()  # 0 -> 1 -> 3 -> None
```

### 双链表的实现

```python
class DoublyListNode:
    """双链表节点"""
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next

class DoublyLinkedList:
    """双链表实现"""
    
    def __init__(self):
        self.head = None
        self.tail = None
        self.size = 0
    
    def add_at_head(self, val):
        """头部添加 - O(1)"""
        new_node = DoublyListNode(val)
        
        if not self.head:
            self.head = self.tail = new_node
        else:
            new_node.next = self.head
            self.head.prev = new_node
            self.head = new_node
        
        self.size += 1
    
    def add_at_tail(self, val):
        """尾部添加 - O(1)"""
        new_node = DoublyListNode(val)
        
        if not self.tail:
            self.head = self.tail = new_node
        else:
            new_node.prev = self.tail
            self.tail.next = new_node
            self.tail = new_node
        
        self.size += 1
    
    def delete_node(self, node):
        """删除节点 - O(1)"""
        if node.prev:
            node.prev.next = node.next
        else:
            self.head = node.next
        
        if node.next:
            node.next.prev = node.prev
        else:
            self.tail = node.prev
        
        self.size -= 1
    
    def display_forward(self):
        """正向显示"""
        elements = []
        current = self.head
        while current:
            elements.append(str(current.val))
            current = current.next
        print(" <-> ".join(elements) + " <-> None")
    
    def display_backward(self):
        """反向显示"""
        elements = []
        current = self.tail
        while current:
            elements.append(str(current.val))
            current = current.prev
        print(" <-> ".join(elements) + " <-> None")

# 测试

dll.add_at_tail(3)
dll.add_at_head(0)
```

```python
def reverse_linked_list(head):
    """
    反转链表
    时间复杂度: O(n)
    空间复杂度: O(1)
    """
    prev = None
    current = head
    
    while current:
        next_temp = current.next  # 保存下一个节点
        current.next = prev       # 反转指针
        prev = current            # 移动 prev
        current = next_temp       # 移动 current
    
    return prev
```

#### 问题 2：检测环

```python
def has_cycle(head):
    """
    检测链表是否有环（快慢指针）
    时间复杂度: O(n)
    空间复杂度: O(1)
    """
    if not head or not head.next:
        return False
    
    slow = head
    fast = head.next
    
    while slow != fast:
        if not fast or not fast.next:
            return False
        slow = slow.next
        fast = fast.next.next
    
    return True

# 创建带环的链表
node1 = ListNode(1)
node2 = ListNode(2)
node3 = ListNode(3)
node4 = ListNode(4)

node1.next = node2
node2.next = node3
node3.next = node4
node4.next = node2  # 形成环

print(f"有环吗? {has_cycle(node1)}")  # True
```

#### 问题 3：合并两个有序链表

```python
def merge_two_lists(l1, l2):
    """
    合并两个有序链表
    时间复杂度: O(m + n)
    空间复杂度: O(1)
    """
    dummy = ListNode(0)
    current = dummy
    
    while l1 and l2:
        if l1.val <= l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next
    
    current.next = l1 if l1 else l2
    
    return dummy.next

# 测试
```

## 2.3 栈（Stack）

### 栈的概念

**栈**是一种后进先出（LIFO - Last In First Out）的线性数据结构。

**基本操作：**
- **push**：入栈，在栈顶添加元素
- **pop**：出栈，移除并返回栈顶元素
- **peek/top**：查看栈顶元素
- **is_empty**：判断栈是否为空

### 使用列表实现栈

```python
class Stack:
    """基于列表的栈实现"""
    
    def __init__(self):
        self.items = []
    
    def push(self, item):
        """入栈 - O(1)"""
        self.items.append(item)
    
    def pop(self):
        """出栈 - O(1)"""
        if self.is_empty():
            raise IndexError("Pop from empty stack")
        return self.items.pop()
    
    def peek(self):
        """查看栈顶 - O(1)"""
        if self.is_empty():
            raise IndexError("Peek from empty stack")
        return self.items[-1]
    
    def is_empty(self):
        """判断是否为空 - O(1)"""
        return len(self.items) == 0
    
    def size(self):
        """获取大小 - O(1)"""
        return len(self.items)
    
    def display(self):
        """显示栈"""
        print(f"栈顶 <- {self.items} <- 栈底")

# 测试
stack = Stack()
stack.push(1)
stack.push(2)
stack.push(3)
stack.display()  # 栈顶 <- [1, 2, 3] <- 栈底

print(f"栈顶元素: {stack.peek()}")  # 3
print(f"出栈: {stack.pop()}")       # 3
stack.display()                      # 栈顶 <- [1, 2] <- 栈底
```

### 栈的应用场景

#### 应用 1：括号匹配

```python
def is_valid_parentheses(s):
    """
    验证括号是否有效
    时间复杂度: O(n)
    空间复杂度: O(n)
    """
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    
    for char in s:
        if char in mapping.values():
            stack.append(char)
        elif char in mapping.keys():
            if not stack or stack.pop() != mapping[char]:
                return False
    
    return not stack

# 测试

print(is_valid_parentheses("()[]{}"))    # True
print(is_valid_parentheses("(]"))        # False
print(is_valid_parentheses("([)]"))      # False
print(is_valid_parentheses("{[]}"))      # True
```

#### 应用 2：表达式求值（逆波兰表达式）

```python
def eval_rpn(tokens):
    """
    计算逆波兰表达式
    时间复杂度: O(n)
    空间复杂度: O(n)
    """
    stack = []
    
    for token in tokens:
        if token in ['+', '-', '*', '/']:
            b = stack.pop()
            a = stack.pop()
            
            if token == '+':
                stack.append(a + b)
            elif token == '-':
                stack.append(a - b)
            elif token == '*':
                stack.append(a * b)
            elif token == '/':
                stack.append(int(a / b))  # 向零取整
        else:
            stack.append(int(token))
    
    return stack[0]

# 测试
# (2 + 1) * 3 = 9
print(eval_rpn(["2", "1", "+", "3", "*"]))  # 9

# (4 + (13 / 5)) = 6
print(eval_rpn(["4", "13", "5", "/", "+"]))  # 6
```

#### 应用 3：最小栈

```python
class MinStack:
    """支持获取最小值的栈"""
    
    def __init__(self):
        self.stack = []
        self.min_stack = []
    
    def push(self, val):
        self.stack.append(val)
        # min_stack 保持最小值
        if not self.min_stack or val <= self.min_stack[-1]:
            self.min_stack.append(val)
    
    def pop(self):
        if self.stack:
            val = self.stack.pop()
            if val == self.min_stack[-1]:
                self.min_stack.pop()
            return val
    
    def top(self):
        return self.stack[-1] if self.stack else None
    
    def get_min(self):
        return self.min_stack[-1] if self.min_stack else None

# 测试
min_stack = MinStack()
min_stack.push(-2)
min_stack.push(0)
min_stack.push(-3)
print(f"最小值: {min_stack.get_min()}")  # -3
min_stack.pop()
print(f"栈顶: {min_stack.top()}")        # 0
print(f"最小值: {min_stack.get_min()}")  # -2
```

## 2.4 队列（Queue）

### 队列的概念

**队列**是一种先进先出（FIFO - First In First Out）的线性数据结构。

**基本操作：**
- **enqueue**：入队，在队尾添加元素
- **dequeue**：出队，移除并返回队首元素
- **front/peek**：查看队首元素
- **is_empty**：判断队列是否为空

### 使用 collections.deque 实现队列

```python
from collections import deque

class Queue:
    """基于 deque 的队列实现"""
    
    def __init__(self):
        self.items = deque()
    
    def enqueue(self, item):
        """入队 - O(1)"""
        self.items.append(item)
    
    def dequeue(self):
        """出队 - O(1)"""
        if self.is_empty():
            raise IndexError("Dequeue from empty queue")
        return self.items.popleft()
    
    def front(self):
        """查看队首 - O(1)"""
        if self.is_empty():
            raise IndexError("Front from empty queue")
        return self.items[0]
    
    def is_empty(self):
        """判断是否为空 - O(1)"""
        return len(self.items) == 0
    
    def size(self):
        """获取大小 - O(1)"""
        return len(self.items)
    
    def display(self):
        """显示队列"""
        print(f"队首 <- {list(self.items)} <- 队尾")

# 测试
queue = Queue()
queue.enqueue(1)
queue.enqueue(2)
queue.enqueue(3)
queue.display()  # 队首 <- [1, 2, 3] <- 队尾

print(f"队首元素: {queue.front()}")  # 1
print(f"出队: {queue.dequeue()}")    # 1
queue.display()                       # 队首 <- [2, 3] <- 队尾
```

### 循环队列

```python
class CircularQueue:
    """循环队列实现"""
    
    def __init__(self, k):
        self.capacity = k
        self.queue = [0] * k
        self.head = 0
        self.tail = 0
        self.size = 0
    
    def enqueue(self, value):
        """入队 - O(1)"""
        if self.is_full():
            return False
        
        self.queue[self.tail] = value
        self.tail = (self.tail + 1) % self.capacity
        self.size += 1
        return True
    
    def dequeue(self):
        """出队 - O(1)"""
        if self.is_empty():
            return False
        
        self.head = (self.head + 1) % self.capacity
        self.size -= 1
        return True
    
    def front(self):
        """查看队首 - O(1)"""
        if self.is_empty():
            return -1
        return self.queue[self.head]
    
    def rear(self):
        """查看队尾 - O(1)"""
        if self.is_empty():
            return -1
        return self.queue[(self.tail - 1) % self.capacity]
    
    def is_empty(self):
        return self.size == 0
    
    def is_full(self):
        return self.size == self.capacity

# 测试
cq = CircularQueue(3)
cq.enqueue(1)
cq.enqueue(2)
cq.enqueue(3)
print(f"队首: {cq.front()}")  # 1
print(f"队尾: {cq.rear()}")   # 3
cq.dequeue()
cq.enqueue(4)
print(f"队尾: {cq.rear()}")   # 4
```

### 优先队列简介

```python
import heapq

class PriorityQueue:
    """基于堆的优先队列（最小堆）"""
    
    def __init__(self):
        self.heap = []
        self.index = 0  # 用于处理相同优先级的情况
    
    def push(self, item, priority):
        """入队 - O(log n)"""
        heapq.heappush(self.heap, (priority, self.index, item))
        self.index += 1
    
    def pop(self):
        """出队（最高优先级） - O(log n)"""
        if self.is_empty():
            raise IndexError("Pop from empty priority queue")
        priority, index, item = heapq.heappop(self.heap)
        return item
    
    def is_empty(self):
        return len(self.heap) == 0
    
    def peek(self):
        """查看最高优先级元素 - O(1)"""
        if self.is_empty():
            return None
        return self.heap[0][2]

# 测试
pq = PriorityQueue()
pq.push("任务C", 3)
pq.push("任务A", 1)
pq.push("任务B", 2)

print(f"最高优先级: {pq.peek()}")  # 任务A
print(f"出队: {pq.pop()}")         # 任务A
print(f"出队: {pq.pop()}")         # 任务B
print(f"出队: {pq.pop()}")         # 任务C
```

### 队列的应用场景

#### 应用 1：广度优先搜索（BFS）

```python
from collections import deque

def bfs_shortest_path(graph, start, end):
    """
    使用 BFS 找最短路径
    时间复杂度: O(V + E)
    空间复杂度: O(V)
    """
    if start == end:
        return [start]
    
    queue = deque([(start, [start])])
    visited = {start}
    
    while queue:
        current, path = queue.popleft()
        
        for neighbor in graph[current]:
            if neighbor not in visited:
                new_path = path + [neighbor]
                
                if neighbor == end:
                    return new_path
                
                visited.add(neighbor)
                queue.append((neighbor, new_path))
    
    return []  # 没有路径

# 测试
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

path = bfs_shortest_path(graph, 'A', 'F')
print(f"最短路径: {' -> '.join(path)}")  # A -> C -> F 或 A -> B -> E -> F
```

#### 应用 2：最近请求次数

```python
class RecentCounter:
    """计算最近 3000 毫秒内的请求数"""
    
    def __init__(self):
        self.queue = deque()
    
    def ping(self, t):
        """
        添加新请求，返回最近 3000ms 内的请求数
        时间复杂度: 均摊 O(1)
        """
        self.queue.append(t)
        
        # 移除过期的请求
        while self.queue[0] < t - 3000:
            self.queue.popleft()
        
        return len(self.queue)

# 测试
counter = RecentCounter()
print(counter.ping(1))     # 1
print(counter.ping(100))   # 2
print(counter.ping(3001))  # 3
print(counter.ping(3002))  # 3
```

## 本章小结

**核心知识点：**

✅ **数组**
- 连续内存，随机访问 O(1)
- 插入删除 O(n)
- Python 列表是动态数组

✅ **链表**
- 单链表、双链表、循环链表
- 插入删除 O(1)（已知位置）
- 访问 O(n)
- 经典问题：反转、检测环、合并

✅ **栈**
- LIFO 原则
- 应用：括号匹配、表达式求值、最小栈

✅ **队列**
- FIFO 原则
- 循环队列、优先队列
- 应用：BFS、任务调度

**选择指南：**

| 场景 | 推荐数据结构 |
|------|-------------|
| 频繁随机访问 | 数组 |
| 频繁插入删除 | 链表 |
| 后进先出 | 栈 |
| 先进先出 | 队列 |
| 优先级处理 | 优先队列 |

```python
class MinStack:
    def __init__(self):
        self.stack = []

    def push(self, x):
        self.stack.append(x)
        if not self.min_stack or x <= self.min_stack[-1]:

    def pop(self):
        if self.stack[-1] == self.min_stack[-1]:
            self.min_stack.pop()
        return self.stack.pop()
```

### 练习 2：用两个栈实现队列

```python
class MyQueue:
    def __init__(self):
        self.in_stack = []
        self.out_stack = []
    
    def push(self, x):
        self.in_stack.append(x)
    
    def pop(self):
        self._move()
        return self.out_stack.pop()
    
    def peek(self):
        self._move()
        return self.out_stack[-1]
    
    def empty(self):
        return not self.in_stack and not self.out_stack
    
    def _move(self):
        if not self.out_stack:
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())
```

```python
def isValid(s):
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    
    for char in s:
        if char in mapping:
            if not stack or stack.pop() != mapping[char]:
```
