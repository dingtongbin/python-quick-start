## 12.1 时间复杂度

### 常见复杂度等级

| 复杂度 | 名称 | 示例 |
|--------|------|------|
| O(1) | 常数 | 数组访问 |
| O(log n) | 对数 | 二分搜索 |
| O(n) | 线性 | 线性搜索 |
| O(n log n) | 线性对数 | 快速排序 |
| O(n²) | 平方 | 冒泡排序 |
| O(n³) | 立方 | 矩阵乘法 |
| O(2ⁿ) | 指数 | 暴力搜索 |
| O(n!) | 阶乘 | 全排列 |

### 复杂度比较

```
n = 10:
O(1) = 1
O(log n) ≈ 3
O(n) = 10
O(n log n) ≈ 30
O(n²) = 100
O(2ⁿ) = 1024
O(n!) = 3,628,800
```

## 12.2 空间复杂度

**定义：** 算法运行所需的额外空间。

```python
# O(1) 空间
def constant_space(arr):
    total = 0
    for num in arr:
        total += num
    return total

# O(n) 空间
def linear_space(arr):
    return [x * 2 for x in arr]
```

## 12.3 数据结构与算法总结

### 数据结构对比

| 数据结构 | 查找 | 插入 | 删除 | 适用场景 |
|----------|------|------|------|----------|
| 数组 | O(n) | O(n) | O(n) | 随机访问 |
| 链表 | O(n) | O(1) | O(1) | 频繁插入删除 |
| 哈希表 | O(1) | O(1) | O(1) | 快速查找 |
| BST | O(log n) | O(log n) | O(log n) | 有序数据 |
| 堆 | O(1) | O(log n) | O(log n) | 优先级队列 |

### 算法对比

| 算法类型 | 代表算法 | 时间复杂度 | 应用场景 |
|----------|----------|------------|----------|
| 排序 | 快速排序 | O(n log n) | 数据排序 |
| 搜索 | 二分搜索 | O(log n) | 有序数据查找 |
| DP | 背包问题 | O(nW) | 最优化问题 |
| 贪心 | 活动选择 | O(n log n) | 局部最优 |
| 回溯 | N 皇后 | O(n!) | 组合问题 |
