本章将系统学习各种排序算法，从基础的冒泡排序到高级的快速排序、归并排序等。排序是计算机科学中最基本也是最重要的算法之一。

## 6.1 排序算法概述

### 什么是排序？

**排序**是将一组数据按照特定顺序（通常是升序或降序）重新排列的过程。

### 排序算法的分类

1. **比较排序**：通过比较元素大小来排序
   - 冒泡排序、选择排序、插入排序
   - 快速排序、归并排序、堆排序

2. **非比较排序**：不通过比较来排序
   - 计数排序、基数排序、桶排序

### 算法性能指标

| 指标 | 说明 |
|------|------|
| 时间复杂度 | 最好、平均、最坏情况 |
| 空间复杂度 | 额外需要的内存空间 |
| 稳定性 | 相等元素的相对顺序是否保持 |
| 原地排序 | 是否需要额外的存储空间 |

## 6.2 基础排序算法

### 1. 冒泡排序（Bubble Sort）

**原理：** 重复遍历列表，比较相邻元素并交换，使较大的元素逐渐"冒泡"到末尾。

```python
def bubble_sort(arr):
    """
    冒泡排序
    时间复杂度: O(n²)
    空间复杂度: O(1)
    稳定性: 稳定
    """
    n = len(arr)
    
    for i in range(n - 1):
        swapped = False
        
        # 最后 i 个元素已经排好序
        for j in range(n - 1 - i):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        
        # 如果没有交换，说明已经有序
        if not swapped:
            break
    
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print("原始数组:", arr)
print("排序后:", bubble_sort(arr.copy()))
```

**可视化过程：**
```
第1轮: [64, 34, 25, 12, 22, 11, 90]
       [34, 25, 12, 22, 11, 64, 90]  ← 64 冒泡到最后

第2轮: [34, 25, 12, 22, 11, 64, 90]
       [25, 12, 22, 11, 34, 64, 90]  ← 34 到位

...以此类推
```

### 2. 选择排序（Selection Sort）

**原理：** 每次从未排序部分找到最小元素，放到已排序部分的末尾。

```python
def selection_sort(arr):
    """
    选择排序
    时间复杂度: O(n²)
    空间复杂度: O(1)
    稳定性: 不稳定
    """
    n = len(arr)
    
    for i in range(n - 1):
        min_idx = i
        
        # 找到最小元素的索引
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        
        # 交换
        if min_idx != i:
            arr[i], arr[min_idx] = arr[min_idx], arr[i]
    
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print("选择排序:", selection_sort(arr.copy()))
```

### 3. 插入排序（Insertion Sort）

**原理：** 将未排序的元素逐个插入到已排序部分的正确位置。

```python
def insertion_sort(arr):
    """
    插入排序
    时间复杂度: O(n²)
    空间复杂度: O(1)
    稳定性: 稳定
    """
    n = len(arr)
    
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        
        # 将比 key 大的元素向后移动
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        
        arr[j + 1] = key
    
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print("插入排序:", insertion_sort(arr.copy()))
```

**插入排序的优势：**
- ✅ 对小数据集效率高
- ✅ 对几乎有序的数据效率接近 O(n)
- ✅ 在线排序（可以边接收数据边排序）

### 基础排序对比

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |

## 6.3 高效排序算法

### 4. 快速排序（Quick Sort）

**原理：** 选择一个基准元素，将数组分为两部分，左边小于基准，右边大于基准，然后递归排序。

```python
def quick_sort(arr):
    """
    快速排序
    时间复杂度: O(n log n) 平均
    空间复杂度: O(log n)
    稳定性: 不稳定
    """
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quick_sort(left) + middle + quick_sort(right)

def quick_sort_inplace(arr, low=0, high=None):
    """快速排序 - 原地排序版本"""
    if high is None:
        high = len(arr) - 1
    
    if low < high:
        # 分区
        pivot_idx = partition(arr, low, high)
        
        # 递归排序左右两部分
        quick_sort_inplace(arr, low, pivot_idx - 1)
        quick_sort_inplace(arr, pivot_idx + 1, high)
    
    return arr

def partition(arr, low, high):
    """分区操作"""
    pivot = arr[high]  # 选择最后一个元素作为基准
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print("快速排序:", quick_sort(arr.copy()))
print("原地快排:", quick_sort_inplace(arr.copy()))
```

**优化技巧：**

```python
def quick_sort_optimized(arr, low=0, high=None):
    """优化的快速排序"""
    if high is None:
        high = len(arr) - 1
    
    while low < high:
        # 小数组使用插入排序
        if high - low < 10:
            insertion_sort_range(arr, low, high)
            return
        
        # 三数取中法选择基准
        mid = (low + high) // 2
        if arr[low] > arr[mid]:
            arr[low], arr[mid] = arr[mid], arr[low]
        if arr[low] > arr[high]:
            arr[low], arr[high] = arr[high], arr[low]
        if arr[mid] > arr[high]:
            arr[mid], arr[high] = arr[high], arr[mid]
        
        arr[mid], arr[high] = arr[high], arr[mid]
        
        # 分区
        pivot_idx = partition(arr, low, high)
        
        # 尾递归优化：先处理较小的部分
        if pivot_idx - low < high - pivot_idx:
            quick_sort_optimized(arr, low, pivot_idx - 1)
            low = pivot_idx + 1
        else:
            quick_sort_optimized(arr, pivot_idx + 1, high)
            high = pivot_idx - 1

def insertion_sort_range(arr, low, high):
    """对指定范围进行插入排序"""
    for i in range(low + 1, high + 1):
        key = arr[i]
        j = i - 1
        while j >= low and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
```

### 5. 归并排序（Merge Sort）

**原理：** 将数组分成两半，分别排序，然后合并。

```python
def merge_sort(arr):
    """
    归并排序
    时间复杂度: O(n log n)
    空间复杂度: O(n)
    稳定性: 稳定
    """
    if len(arr) <= 1:
        return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)

def merge(left, right):
    """合并两个有序数组"""
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    
    return result

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print("归并排序:", merge_sort(arr))
```

**可视化过程：**
```
[64, 34, 25, 12, 22, 11, 90]
     ↓ 分割
[64, 34, 25]  [12, 22, 11, 90]
   ↓              ↓
[64] [34, 25]  [12, 22] [11, 90]
      ↓           ↓        ↓
[64] [25, 34]  [12, 22] [11, 90]
   ↓              ↓        ↓
[25, 34, 64]  [12, 22, 11, 90]
        ↓
[11, 12, 22, 25, 34, 64, 90]
```

### 6. 堆排序（Heap Sort）

**原理：** 利用堆这种数据结构进行排序。

```python
def heap_sort(arr):
    """
    堆排序
    时间复杂度: O(n log n)
    空间复杂度: O(1)
    稳定性: 不稳定
    """
    n = len(arr)
    
    # 构建最大堆
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    
    # 逐个提取元素
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]  # 交换
        heapify(arr, i, 0)  # 调整堆
    
    return arr

def heapify(arr, n, i):
    """维护堆的性质"""
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2
    
    if left < n and arr[left] > arr[largest]:
        largest = left
    
    if right < n and arr[right] > arr[largest]:
        largest = right
    
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print("堆排序:", heap_sort(arr.copy()))
```

### 高效排序对比

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) | 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 不稳定 |

## 6.4 特殊排序算法

### 7. 计数排序（Counting Sort）

适用于**整数**且**范围已知**的情况。

```python
def counting_sort(arr):
    """
    计数排序
    时间复杂度: O(n + k)，k 是范围
    空间复杂度: O(k)
    稳定性: 稳定
    """
    if not arr:
        return []
    
    min_val = min(arr)
    max_val = max(arr)
    range_val = max_val - min_val + 1
    
    # 计数
    count = [0] * range_val
    for num in arr:
        count[num - min_val] += 1
    
    # 重建数组
    result = []
    for i in range(range_val):
        result.extend([i + min_val] * count[i])
    
    return result

# 测试
arr = [4, 2, 2, 8, 3, 3, 1]
print("计数排序:", counting_sort(arr))
```

### 8. 基数排序（Radix Sort）

适用于**整数**，按位排序。

```python
def radix_sort(arr):
    """
    基数排序
    时间复杂度: O(d * (n + k))，d 是位数
    空间复杂度: O(n + k)
    稳定性: 稳定
    """
    if not arr:
        return []
    
    max_val = max(arr)
    exp = 1
    
    while max_val // exp > 0:
        counting_sort_by_digit(arr, exp)
        exp *= 10
    
    return arr

def counting_sort_by_digit(arr, exp):
    """按指定位排序"""
    n = len(arr)
    output = [0] * n
    count = [0] * 10
    
    # 计数
    for num in arr:
        index = num // exp % 10
        count[index] += 1
    
    # 累加
    for i in range(1, 10):
        count[i] += count[i - 1]
    
    # 构建输出数组（从后往前保证稳定性）
    for i in range(n - 1, -1, -1):
        index = arr[i] // exp % 10
        output[count[index] - 1] = arr[i]
        count[index] -= 1
    
    # 复制回原数组
    for i in range(n):
        arr[i] = output[i]

# 测试
arr = [170, 45, 75, 90, 802, 24, 2, 66]
print("基数排序:", radix_sort(arr.copy()))
```

### 9. 桶排序（Bucket Sort）

适用于**均匀分布**的数据。

```python
def bucket_sort(arr, bucket_size=10):
    """
    桶排序
    时间复杂度: O(n + k) 平均
    空间复杂度: O(n + k)
    稳定性: 稳定
    """
    if not arr:
        return []
    
    min_val = min(arr)
    max_val = max(arr)
    
    # 创建桶
    bucket_count = (max_val - min_val) // bucket_size + 1
    buckets = [[] for _ in range(bucket_count)]
    
    # 分配到桶中
    for num in arr:
        index = (num - min_val) // bucket_size
        buckets[index].append(num)
    
    # 对每个桶排序并合并
    result = []
    for bucket in buckets:
        insertion_sort(bucket)
        result.extend(bucket)
    
    return result

# 测试
arr = [0.42, 0.32, 0.23, 0.52, 0.25, 0.47, 0.51]
print("桶排序:", bucket_sort(arr))
```

### 练习 2：寻找第 K 大元素

使用快速排序的思想，在 O(n) 时间内找到第 K 大的元素。

```python
def find_kth_largest(nums, k):
    """快速选择算法"""
    def quick_select(left, right, k_smallest):
        if left == right:
            return nums[left]
        
        pivot_index = partition(left, right)
        
        if k_smallest == pivot_index:
            return nums[k_smallest]
        elif k_smallest < pivot_index:
            return quick_select(left, pivot_index - 1, k_smallest)
        else:
            return quick_select(pivot_index + 1, right, k_smallest)
    
    return quick_select(0, len(nums) - 1, len(nums) - k)
```

### 练习 3：合并 K 个有序链表

使用归并排序的思想合并 K 个有序链表。
