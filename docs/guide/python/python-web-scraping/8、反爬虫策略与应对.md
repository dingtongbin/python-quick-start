## 8.1 常见反爬手段

### 1. User-Agent 检测

**检测方式：** 检查请求头中的 User-Agent

**应对：** 使用真实浏览器的 UA，定期轮换

```python
from fake_useragent import UserAgent

ua = UserAgent()
headers = {'User-Agent': ua.random}
```

### 2. IP 频率限制

**检测方式：** 同一 IP 短时间内大量请求

**应对：** 
- 降低请求频率
- 使用代理池
- 分布式爬取

### 3. Cookie 验证

**检测方式：** 检查是否携带有效 Cookie

**应对：** 
- 使用 Session
- 模拟登录获取 Cookie
- 保存和复用 Cookie

### 4. JavaScript 挑战

**检测方式：** 执行 JS 生成 token

**应对：** 
- 使用 Selenium/Playwright
- 逆向 JS 算法
- 使用 PyExecJS

### 5. 验证码

**检测方式：** 弹出验证码

**应对：** 
- OCR 识别
- 打码平台
- 人工识别

### 6. 浏览器指纹

**检测方式：** 检测 Canvas、WebGL 等特征

**应对：** 
- 使用真实浏览器
- 修改指纹参数

## 8.3 IP 代理池

### 代理池实现

```python
import requests
import random
import time

class ProxyPool:
    """代理池"""
    
    def __init__(self):
        self.proxies = []
        self.failed_proxies = set()
        self.last_update = 0
        self.update_interval = 300  # 5 分钟更新一次
    
    def update_proxies(self):
        """从代理网站获取代理"""
        try:
            response = requests.get('https://free-proxy-list.net/', timeout=10)
            # 解析代理列表
            # ...
            self.proxies = [...]  # 新代理列表
            self.last_update = time.time()
        except Exception as e:
            print(f"更新代理失败: {e}")
    
    def get_proxy(self):
        """获取可用代理"""
        # 检查是否需要更新
        if time.time() - self.last_update > self.update_interval:
            self.update_proxies()
        
        # 过滤失效代理
        available = [p for p in self.proxies if p not in self.failed_proxies]
        
        if not available:
            self.failed_proxies.clear()
            available = self.proxies
        
        return random.choice(available) if available else None
    
    def mark_failed(self, proxy):
        """标记代理失败"""
        self.failed_proxies.add(proxy)
        if len(self.failed_proxies) > len(self.proxies) * 0.5:
            self.update_proxies()

# Scrapy 中间件
class ProxyMiddleware:
    def __init__(self):
        self.pool = ProxyPool()
    
    def process_request(self, request, spider):
        proxy = self.pool.get_proxy()
        if proxy:
            request.meta['proxy'] = proxy
```

## 8.4 请求频率控制

### 随机延迟

```python
import time
import random

def random_delay(min_delay=1, max_delay=3):
    """随机延迟"""
    delay = random.uniform(min_delay, max_delay)
    time.sleep(delay)
```

```python
import pytesseract
from PIL import Image
import io

def recognize_captcha(image_data):
    """识别验证码"""
    image = Image.open(io.BytesIO(image_data))
    
    # 预处理
    image = image.convert('L')  # 灰度
    image = image.point(lambda x: 0 if x < 128 else 255, '1')  # 二值化
    
    # OCR
    text = pytesseract.image_to_string(image)
    
    return text.strip()
```

```python
import requests

class CaptchaSolver:
    """打码平台"""
    
    def __init__(self, api_key):

        """识别验证码"""
        files = {'userfile': ('captcha.jpg', image_data)}
```

## 8.8 实战：完整反爬方案

### 综合爬虫

```python
import scrapy
import random
import time
from fake_useragent import UserAgent

class AntiBlockSpider(scrapy.Spider):
    name = 'anti_block'
    
    custom_settings = {
        'DOWNLOAD_DELAY': 2,
        'RANDOMIZE_DOWNLOAD_DELAY': True,
        'CONCURRENT_REQUESTS_PER_DOMAIN': 2,
        'COOKIES_ENABLED': True,
        'RETRY_TIMES': 3,
        'RETRY_HTTP_CODES': [403, 429, 500, 502, 503, 504],
    }
    
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self.ua = UserAgent()
        self.proxy_pool = ProxyPool()
    
    def start_requests(self):
        urls = ['http://example.com/page1', ...]
        
        for url in urls:
            yield scrapy.Request(
                url=url,
                callback=self.parse,
                headers={'User-Agent': self.ua.random},
                meta={'proxy': self.proxy_pool.get_proxy()},
                errback=self.handle_error
            )
    
    def parse(self, response):
        # 正常解析
        yield {...}
        
        # 随机延迟
        time.sleep(random.uniform(1, 3))
    
    def handle_error(self, failure):
        """错误处理"""
        request = failure.request
        
        if failure.check(scrapy.http.Response):
            response = failure.value
            if response.status == 403:
                self.logger.warning(f"被禁止: {request.url}")
                # 切换代理重试
                request.meta['proxy'] = self.proxy_pool.get_proxy()
                yield request
            elif response.status == 429:
                self.logger.warning(f"频率限制: {request.url}")
                # 等待后重试
                time.sleep(10)
                yield request
```
