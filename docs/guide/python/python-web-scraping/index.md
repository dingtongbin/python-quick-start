## 🛠️ 技术栈

### 核心库
- **requests** - HTTP 请求
- **BeautifulSoup4** - HTML 解析
- **lxml** - XML/HTML 解析
- **Selenium** - 浏览器自动化
- **aiohttp** - 异步 HTTP
- **Scrapy** - 爬虫框架

### 数据存储
- **pymysql** - MySQL 驱动
- **pymongo** - MongoDB 驱动
- **redis** - Redis 客户端
- **elasticsearch** - ES 客户端

### 工具
- **fake-useragent** - UA 生成
- **pyppeteer** - 无头浏览器
- **celery** - 任务队列
- **docker** - 容器化

## ⚠️ 重要声明

**本教程仅用于教育和学习目的！**

- ✅ 遵守目标网站的 robots.txt
- ✅ 尊重网站的服务条款
- ✅ 控制请求频率，避免影响网站正常运行
- ✅ 不抓取个人隐私数据
- ✅ 不用于商业用途（除非获得授权）
- ❌ 不要用于非法活动
- ❌ 不要攻击或破坏网站
