本章将学习如何将 PySide6 应用打包成可执行文件，方便分发和部署。

## 11.1 PyInstaller 简介

### 什么是 PyInstaller？

**PyInstaller** 是一个将 Python 程序打包成独立可执行文件的工具，支持 Windows、macOS、Linux。

**特点：**
- ✅ 无需安装 Python 即可运行
- ✅ 包含所有依赖
- ✅ 支持单文件和目录模式
- ✅ 跨平台支持

### 安装

```bash
pip install pyinstaller
```

---

## 11.2 基本打包

### 快速开始

```bash
# 最简单的打包命令
pyinstaller your_script.py

# 生成文件在 dist/your_script/ 目录
```

### 常用参数

```bash
pyinstaller \
    --name="MyApp" \              # 应用名称
    --onefile \                   # 打包成单个exe文件
    --windowed \                  # 无控制台窗口（GUI应用）
    --icon=app.ico \             # 应用图标
    --add-data="config.ini;." \  # 添加文件（Windows用分号）
    --add-data="images:images" \ # 添加文件夹（macOS/Linux用冒号）
    --hidden-import=module \     # 隐藏导入
    --exclude-module=test \      # 排除模块
    --clean \                     # 清理缓存
    your_script.py
```

### 参数详解

| 参数 | 说明 | 示例 |
|------|------|------|
| `--name` | 应用名称 | `--name="MyApp"` |
| `--onefile` | 单文件模式 | 生成单个exe |
| `--onedir` | 目录模式（默认） | 生成文件夹 |
| `--windowed` | 无控制台 | GUI应用使用 |
| `--console` | 有控制台（默认） | 命令行应用 |
| `--icon` | 图标文件 | `--icon=app.ico` |
| `--add-data` | 添加数据文件 | `--add-data="file;dest"` |
| `--hidden-import` | 隐藏导入 | 自动检测不到的模块 |
| `--exclude-module` | 排除模块 | 减小体积 |
| `--paths` | 添加搜索路径 | `--paths=/path/to/libs` |
| `--clean` | 清理缓存 | 重新构建 |

## 11.3 打包 Windows 应用

### 基本打包

```bash
# 单文件模式
pyinstaller --onefile --windowed --name="TodoApp" main.py

# 带图标
pyinstaller --onefile --windowed --icon=app.ico --name="TodoApp" main.py
```

### 添加资源文件

```bash
# 添加配置文件
pyinstaller --onefile --windowed \
    --add-data="config.ini;." \
    --add-data="templates;templates" \
    --name="MyApp" \
    main.py
```

### 创建 ICO 图标

```python
# 使用 PIL 创建图标
from PIL import Image

# 打开 PNG 图片
img = Image.open('logo.png')

# 保存为 ICO（支持多个尺寸）
img.save('app.ico', format='ICO', sizes=[(16, 16), (32, 32), (48, 48), (64, 64)])
```

## 11.6 Spec 文件配置

### 什么是 Spec 文件？

Spec 文件是 PyInstaller 的配置文件，可以更精细地控制打包过程。

### 自动生成

```bash
# 首次运行时会自动生成 .spec 文件
pyinstaller main.py

# 之后可以直接使用 spec 文件
pyinstaller MyApp.spec
```

### 手动编写 Spec 文件

```python
# myapp.spec
# -*- mode: python ; coding: utf-8 -*-

block_cipher = None

# 分析阶段
a = Analysis(
    ['main.py'],                          # 入口脚本
    pathex=[],                            # 额外搜索路径
    binaries=[],                          # 二进制文件
    datas=[                               # 数据文件
        ('config.ini', '.'),
        ('images', 'images'),
        ('templates', 'templates'),
    ],
    hiddenimports=[                       # 隐藏导入
        'pkg_resources.py2_warn',
    ],
    hookspath=[],                         # hooks 路径
    runtime_hooks=[],                     # 运行时 hooks
    excludes=[                            # 排除模块
        'matplotlib',
        'tkinter',
        'unittest',
    ],
    win_no_prefer_redirects=False,
    win_private_assemblies=False,
    cipher=block_cipher,
    noarchive=False,                      # 不归档（调试用）
)

# 打包 Python 字节码
pyz = PYZ(a.pure, a.zipped_data, cipher=block_cipher)

# 创建可执行文件
exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.zipfiles,
    a.datas,
    [],                                   # 额外文件
    name='MyApp',                         # 输出名称
    debug=False,                          # 调试模式
    bootloader_ignore_signals=False,
    strip=True,                           # 去除符号表
    upx=True,                             # 使用 UPX 压缩
    upx_exclude=[],                       # UPX 排除的文件
    runtime_tmpdir=None,
    console=False,                        # 无控制台
    disable_windowed_traceback=False,
    target_arch=None,
    codesign_identity=None,
    entitlements_file=None,
)

# macOS 应用 bundle
app = BUNDLE(
    exe,
    name='MyApp.app',
    icon='app.icns',
    bundle_identifier='com.example.myapp',
)
```

### 使用 Spec 文件

```bash
pyinstaller MyApp.spec
```

## 11.7 优化打包体积

### 常见问题

❌ **打包后文件太大**（几十MB甚至几百MB）

### 优化方法

#### 1. 排除不需要的模块

```python
# 在 spec 文件中
excludes=[
    'matplotlib',      # 如果不用
    'numpy',           # 如果不用
    'pandas',          # 如果不用
    'tkinter',         # PySide6 不需要
    'unittest',
    'doctest',
    'pdb',
]
```

#### 2. 使用 UPX 压缩

```bash
# 安装 UPX
# Windows: 下载 https://github.com/upx/upx/releases
# macOS: brew install upx
# Linux: sudo apt install upx

# 使用 UPX
pyinstaller --upx-dir=/path/to/upx --onefile main.py
```

#### 3. 清理缓存

```bash
# 删除 build 和 dist 目录
rm -rf build dist *.spec

# 重新打包
pyinstaller --clean main.py
```

#### 4. 只导入需要的模块

```python
# ❌ 导入整个库
import matplotlib

# ✅ 只导入需要的部分
from matplotlib import pyplot as plt
```

#### 5. 使用虚拟环境

```bash
# 创建干净的虚拟环境
python -m venv clean_env
source clean_env/bin/activate  # Linux/macOS
# clean_env\Scripts\activate  # Windows

# 只安装必要的包
pip install PySide6

# 打包
pyinstaller main.py
```

### 体积对比

| 优化前 | 优化后 | 方法 |
|--------|--------|------|
| 200 MB | 50 MB | 排除不必要模块 |
| 50 MB | 30 MB | 使用 UPX 压缩 |
| 30 MB | 25 MB | 清理缓存 |

#### 问题 2：资源文件找不到

```python
# 获取资源文件路径
import sys
import os

def resource_path(relative_path):
    """获取资源文件的绝对路径"""
    if getattr(sys, 'frozen', False):
        # PyInstaller 打包后
        base_path = sys._MEIPASS
    else:
        # 开发环境
        base_path = os.path.abspath(".")
    
    return os.path.join(base_path, relative_path)

# 使用
config_file = resource_path('config.ini')
```

## 11.9 自动化打包脚本

### Windows batch 脚本

```batch
@echo off
echo 开始打包...

REM 清理旧文件
rmdir /s /q build dist
del *.spec

REM 打包
pyinstaller --onefile --windowed --name="MyApp" --icon=app.ico main.py

echo 打包完成！
pause
```

### macOS/Linux shell 脚本

```bash
#!/bin/bash

echo "开始打包..."

# 清理旧文件
rm -rf build dist *.spec

# 打包
pyinstaller --onefile --windowed --name="MyApp" --icon=app.icns main.py

echo "打包完成！"
```

### Python 脚本

```python
import subprocess
import shutil
import os

def build_app():
    """自动化打包"""
    
    # 清理
    print("清理旧文件...")
    for dir_name in ['build', 'dist']:
        if os.path.exists(dir_name):
            shutil.rmtree(dir_name)
    
    for file in os.listdir('.'):
        if file.endswith('.spec'):
            os.remove(file)
    
    # 打包
    print("开始打包...")
    cmd = [
        'pyinstaller',
        '--onefile',
        '--windowed',
        '--name=MyApp',
        '--icon=app.ico',
        '--clean',
        'main.py'
    ]
    
    subprocess.run(cmd, check=True)
    
    print("打包完成！")
    print("可执行文件在 dist/ 目录")

if __name__ == '__main__':
    build_app()
```
