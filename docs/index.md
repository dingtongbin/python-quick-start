## 💻 本地运行

如果你想在本机预览文档，可以按照以下步骤操作：

```bash
git clone https://github.com/dingtongbin/python-quick-start.git
cd python-quick-start
npm install
npm run docs:dev
```

启动后在浏览器中访问 `http://localhost:5173/python-quick-start/` 就能看到文档了。
