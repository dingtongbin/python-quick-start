## ✨ 这是什么？

这是一份面向 **零基础** 和 **转行开发者** 的 Python 系统教程，采用 [VitePress](https://vitepress.dev/) 构建，内容从基础语法到实战项目，覆盖 6 大方向，共 **85+ 篇** 详细教程。

所有示例基于 **Python 3.10**，强调动手实践，拒绝纸上谈兵。

## 🚀 快速开始

### 推荐学习路线

```
Python 基础 → 数据分析 / 网络爬虫（选一个）
                ↓
        数据结构与算法（面试必备）
                ↓
    Django Web 开发 / PySide6 桌面开发（选一个）
```

### 本地运行

```bash
# 克隆项目
git clone https://github.com/dingtongbin/python-quick-start.git
cd python-quick-start

# 安装依赖
npm install

# 启动本地开发服务器
npm run docs:dev

# 构建静态站点
npm run docs:build

# 预览构建结果
npm run docs:preview
```
